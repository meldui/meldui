# add-content-protection

Add an opt-in `contentProtection` flag to the DocumentViewer: a bundle of client-side capture **deterrents** (obscure-on-blur/tab-hide, best-effort PrintScreen clipboard clobber, block right-click + drag-out). A deterrent against casual capture only — not a guarantee, and explicitly not DRM. Does not touch copy/print/download (those keep their existing flags).
