# Row Selection

The DataTable supports row selection with checkboxes. Selected rows can be used for bulk actions like exporting, deleting, or updating multiple records at once.

## Enabling Selection

Two things are required to enable row selection:

1. Add a `selection()` column to your column definitions
2. Set `enableRowSelection` to `true` on the DataTable component

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { DataTable, createColumnHelper } from '@meldui/vue'

interface Task {
  id: number
  title: string
  status: string
  assignee: string
}

const helper = createColumnHelper<Task>()

const columns = [
  helper.selection(), // Add checkbox column
  helper.accessor('title', { title: 'Title' }),
  helper.accessor('status', { title: 'Status' }),
  helper.accessor('assignee', { title: 'Assignee' }),
]

const data = ref<Task[]>([])
const pageCount = ref(0)

async function handleServerSideChange({ sorting, filters, pagination }) {
  // fetch data...
}
</script>

<template>
  <DataTable
    :columns="columns"
    :data="data"
    :page-count="pageCount"
    :on-server-side-change="handleServerSideChange"
    :enable-row-selection="true"
  />
</template>
```

## Selection Behavior

The selection column renders:

- **Header checkbox**: Toggles selection of all rows on the current page. Shows an indeterminate state when only some rows are selected.
- **Row checkbox**: Toggles selection of an individual row.

Selection state is tracked per page. Navigating to a new page does not clear the selection from previous pages -- TanStack Table maintains the selection using row indices.

## Showing Selected Count

To display how many rows are selected in the pagination area:

```vue
<DataTable
  :columns="columns"
  :data="data"
  :page-count="pageCount"
  :on-server-side-change="handleServerSideChange"
  :enable-row-selection="true"
  :show-selected-count="true"
/>
```

## Accessing Selected Rows

Use a template ref to access the exposed `selectedRows` computed property:

```vue
<script setup lang="ts">
import { ref, watch } from 'vue'
import { DataTable } from '@meldui/vue'

const tableRef = ref<InstanceType<typeof DataTable>>()

// React to selection changes
watch(
  () => tableRef.value?.selectedRows,
  (rows) => {
    if (rows) {
      console.log('Selected:', rows)
    }
  },
  { deep: true },
)

function handleExport() {
  const selected = tableRef.value?.selectedRows ?? []
  exportToCsv(selected)
}
</script>

<template>
  <DataTable ref="tableRef" :enable-row-selection="true" ... />
  <button @click="handleExport">Export Selected</button>
</template>
```

The exposed selection-related properties:

| Property           | Type                     | Description                                          |
| ------------------ | ------------------------ | ---------------------------------------------------- |
| `rowSelection`     | `Ref<RowSelectionState>` | Raw selection state (row index to boolean map)       |
| `selectedRows`     | `ComputedRef<TData[]>`   | Array of the original data objects for selected rows |
| `selectedRowCount` | `ComputedRef<number>`    | Number of selected rows                              |
| `hasSelection`     | `ComputedRef<boolean>`   | Whether any rows are currently selected              |
| `resetSelection()` | `() => void`             | Clear all row selections                             |

## Bulk Actions

Bulk actions let you define a dropdown menu of operations that act on the currently selected rows. The dropdown automatically appears in the toolbar when rows are selected.

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { DataTable, createColumnHelper } from '@meldui/vue'
import type { BulkActionOption } from '@meldui/vue'
import { IconDownload, IconTrash, IconArchive } from '@meldui/tabler-vue'

interface Task {
  id: number
  title: string
  status: string
}

const helper = createColumnHelper<Task>()

const columns = [
  helper.selection(),
  helper.accessor('title', { title: 'Title' }),
  helper.accessor('status', { title: 'Status' }),
]

const bulkActions: BulkActionOption<Task>[] = [
  {
    label: 'Export Selected',
    icon: IconDownload,
    action: (selectedRows) => {
      exportToCsv(selectedRows)
    },
  },
  {
    label: 'Archive Selected',
    icon: IconArchive,
    action: (selectedRows) => {
      archiveTasks(selectedRows.map((row) => row.id))
    },
  },
  {
    label: 'Delete Selected',
    icon: IconTrash,
    variant: 'destructive',
    action: (selectedRows) => {
      if (confirm(`Delete ${selectedRows.length} tasks?`)) {
        deleteTasks(selectedRows.map((row) => row.id))
      }
    },
  },
]

const data = ref<Task[]>([])
const pageCount = ref(0)

async function handleServerSideChange({ sorting, filters, pagination }) {
  // fetch data...
}
</script>

<template>
  <DataTable
    :columns="columns"
    :data="data"
    :page-count="pageCount"
    :on-server-side-change="handleServerSideChange"
    :enable-row-selection="true"
    :show-selected-count="true"
    :bulk-select-options="bulkActions"
  />
</template>
```

### BulkActionOption Interface

| Property  | Type                              | Description                                               |
| --------- | --------------------------------- | --------------------------------------------------------- |
| `label`   | `string`                          | Action label displayed in the dropdown menu               |
| `icon`    | `Component`                       | Optional icon component displayed before the label        |
| `variant` | `'default' \| 'destructive'`      | Visual style; `destructive` renders the text in red       |
| `action`  | `(selectedRows: TData[]) => void` | Callback receiving the array of selected row data objects |

The bulk actions dropdown also includes a "Clear" option at the bottom that deselects all rows.

## Selection with Keyboard Navigation

When keyboard navigation is enabled alongside selection, the Space key toggles selection of the focused row:

```vue
<DataTable
  :columns="columns"
  :data="data"
  :page-count="pageCount"
  :on-server-side-change="handleServerSideChange"
  :enable-row-selection="true"
  :enable-keyboard-navigation="true"
/>
```

Keyboard shortcuts for selection:

| Key      | Action                                   |
| -------- | ---------------------------------------- |
| `Space`  | Toggle selection of the focused row      |
| `Escape` | Clear all selections and reset row focus |

## Pinning the Selection Column

It is common to pin the selection checkbox column to the left edge so it remains visible while scrolling horizontally:

```vue
<DataTable
  :columns="columns"
  :data="data"
  :page-count="pageCount"
  :on-server-side-change="handleServerSideChange"
  :enable-row-selection="true"
  :enable-column-pinning="true"
  :default-pinning="{ left: ['select'], right: [] }"
/>
```

The selection column has `enablePinning: true` by default, so it works seamlessly with column pinning. See the [Column Pinning](/data-table/column-pinning) page for more details.
