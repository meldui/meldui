title: Data Display

Data display blocks are page-level patterns for presenting information to users. They cover stat card grids, list views for browsing items, and detail views for inspecting a single record. These patterns use Card, Table, Badge, Avatar, Progress, and Separator to present data clearly.

## Stat Card Grid

A row of metric cards that summarize key data points. This is the same pattern used in dashboards but can appear on any page where you need a quick numeric overview.

```vue
<script setup lang="ts">
import {
  IconArrowDownRight,
  IconArrowUpRight,
  IconCash,
  IconShoppingCart,
  IconUsers,
  IconEye,
} from '@meldui/tabler-vue'
import { Card, CardContent, CardHeader, CardTitle } from '@meldui/vue'
import type { Component } from 'vue'

interface StatCard {
  title: string
  value: string
  change: string
  changeType: 'positive' | 'negative'
  icon: Component
}

const stats: StatCard[] = [
  {
    title: 'Total Revenue',
    value: '$45,231.89',
    change: '+20.1%',
    changeType: 'positive',
    icon: IconCash,
  },
  {
    title: 'Subscriptions',
    value: '+2,350',
    change: '+180.1%',
    changeType: 'positive',
    icon: IconUsers,
  },
  {
    title: 'Sales',
    value: '+12,234',
    change: '+19%',
    changeType: 'positive',
    icon: IconShoppingCart,
  },
  { title: 'Page Views', value: '573,281', change: '-2.4%', changeType: 'negative', icon: IconEye },
]
</script>

<template>
  <div class="grid gap-4 md:grid-cols-2 lg:grid-cols-4">
    <Card v-for="stat in stats" :key="stat.title">
      <CardHeader class="flex flex-row items-center justify-between space-y-0 pb-2">
        <CardTitle class="text-sm font-medium">{{ stat.title }}</CardTitle>
        <component :is="stat.icon" class="h-4 w-4 text-muted-foreground" />
      </CardHeader>
      <CardContent>
        <div class="text-2xl font-bold">{{ stat.value }}</div>
        <div class="flex items-center text-xs mt-1">
          <component
            :is="stat.changeType === 'positive' ? IconArrowUpRight : IconArrowDownRight"
            class="h-3 w-3 mr-1"
            :class="stat.changeType === 'positive' ? 'text-green-600' : 'text-red-600'"
          />
          <span :class="stat.changeType === 'positive' ? 'text-green-600' : 'text-red-600'">
            {{ stat.change }}
          </span>
          <span class="text-muted-foreground ml-1">from last month</span>
        </div>
      </CardContent>
    </Card>
  </div>
</template>
```

## List View

A list view shows a collection of items in a vertical layout with summary information and actions. Each item is a row inside a Card, separated by borders. This pattern works well for emails, notifications, or any list of homogeneous records.

```vue
<script setup lang="ts">
import { IconDots } from '@meldui/tabler-vue'
import {
  Avatar,
  AvatarFallback,
  Badge,
  Button,
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
  RelativeTime,
  Separator,
} from '@meldui/vue'

interface Member {
  id: string
  name: string
  email: string
  role: 'admin' | 'editor' | 'viewer'
  joinedAt: string
  initials: string
}

const members: Member[] = [
  {
    id: '1',
    name: 'Alice Chen',
    email: 'alice@example.com',
    role: 'admin',
    joinedAt: '2024-01-15',
    initials: 'AC',
  },
  {
    id: '2',
    name: 'Bob Smith',
    email: 'bob@example.com',
    role: 'editor',
    joinedAt: '2024-03-22',
    initials: 'BS',
  },
  {
    id: '3',
    name: 'Carol Jones',
    email: 'carol@example.com',
    role: 'viewer',
    joinedAt: '2024-06-10',
    initials: 'CJ',
  },
  {
    id: '4',
    name: 'David Kim',
    email: 'david@example.com',
    role: 'editor',
    joinedAt: '2024-08-05',
    initials: 'DK',
  },
]

const getRoleBadgeVariant = (role: string) => {
  switch (role) {
    case 'admin':
      return 'default'
    case 'editor':
      return 'secondary'
    default:
      return 'outline'
  }
}
</script>

<template>
  <Card>
    <CardHeader>
      <CardTitle>Team Members</CardTitle>
      <CardDescription>Manage your team and their roles.</CardDescription>
    </CardHeader>
    <CardContent class="p-0">
      <div v-for="(member, index) in members" :key="member.id">
        <div class="flex items-center justify-between px-6 py-4">
          <div class="flex items-center gap-4">
            <Avatar>
              <AvatarFallback>{{ member.initials }}</AvatarFallback>
            </Avatar>
            <div>
              <p class="text-sm font-medium">{{ member.name }}</p>
              <p class="text-xs text-muted-foreground">{{ member.email }}</p>
            </div>
          </div>
          <div class="flex items-center gap-4">
            <Badge :variant="getRoleBadgeVariant(member.role)" class="capitalize">
              {{ member.role }}
            </Badge>
            <span class="text-xs text-muted-foreground hidden sm:block">
              Joined <RelativeTime :date="member.joinedAt" />
            </span>
            <DropdownMenu>
              <DropdownMenuTrigger as-child>
                <Button variant="ghost" size="icon" class="h-8 w-8">
                  <IconDots :size="16" />
                </Button>
              </DropdownMenuTrigger>
              <DropdownMenuContent align="end">
                <DropdownMenuItem>Edit role</DropdownMenuItem>
                <DropdownMenuItem>View profile</DropdownMenuItem>
                <DropdownMenuItem class="text-destructive">Remove</DropdownMenuItem>
              </DropdownMenuContent>
            </DropdownMenu>
          </div>
        </div>
        <Separator v-if="index < members.length - 1" />
      </div>
    </CardContent>
  </Card>
</template>
```

## Detail View

A detail view shows all the information about a single record. Use a two-column layout with a summary sidebar on the left and detailed content on the right (or stack them on mobile).

```vue
<script setup lang="ts">
import { IconCalendar, IconTag, IconUser } from '@meldui/tabler-vue'
import {
  Avatar,
  AvatarFallback,
  Badge,
  Button,
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
  Progress,
  Separator,
} from '@meldui/vue'

const project = {
  name: 'Website Redesign',
  description:
    'Complete overhaul of the marketing website with new branding, updated content, and improved performance.',
  status: 'In Progress',
  priority: 'High',
  progress: 65,
  dueDate: 'March 30, 2025',
  owner: { name: 'Alice Chen', initials: 'AC' },
  tags: ['design', 'frontend', 'marketing'],
  tasks: { total: 24, completed: 16 },
}
</script>

<template>
  <div class="grid gap-6 md:grid-cols-3">
    <!-- Main Content -->
    <div class="md:col-span-2 space-y-6">
      <Card>
        <CardHeader>
          <div class="flex items-center justify-between">
            <div>
              <CardTitle class="text-2xl">{{ project.name }}</CardTitle>
              <CardDescription class="mt-2">{{ project.description }}</CardDescription>
            </div>
            <Button>Edit</Button>
          </div>
        </CardHeader>
        <CardContent>
          <div class="space-y-4">
            <div>
              <p class="text-sm font-medium mb-2">Progress</p>
              <Progress :model-value="project.progress" class="h-2" />
              <p class="text-xs text-muted-foreground mt-1">
                {{ project.tasks.completed }} of {{ project.tasks.total }} tasks completed ({{
                  project.progress
                }}%)
              </p>
            </div>
          </div>
        </CardContent>
      </Card>

      <!-- Activity or sub-content can go here -->
      <Card>
        <CardHeader>
          <CardTitle>Recent Activity</CardTitle>
        </CardHeader>
        <CardContent>
          <p class="text-sm text-muted-foreground">Activity timeline would go here.</p>
        </CardContent>
      </Card>
    </div>

    <!-- Sidebar -->
    <div class="space-y-6">
      <Card>
        <CardHeader>
          <CardTitle>Details</CardTitle>
        </CardHeader>
        <CardContent class="space-y-4">
          <div class="flex items-center gap-3">
            <IconUser class="h-4 w-4 text-muted-foreground" />

              <p class="text-xs text-muted-foreground">Owner</p>

                <Avatar class="h-6 w-6">
                  <AvatarFallback class="text-xs">{{ project.owner.initials }}</AvatarFallback>
                </Avatar>
                <span class="text-sm">{{ project.owner.name }}</span>
              </div>
            </div>
          </div>
          <Separator />

          <div class="flex items-center gap-3">
            <IconCalendar class="h-4 w-4 text-muted-foreground" />
            <div>
              <p class="text-xs text-muted-foreground">Due Date</p>
              <p class="text-sm">{{ project.dueDate }}</p>
            </div>
          </div>
          <Separator />

          <div>
            <div class="flex items-center gap-3 mb-2">
              <IconTag class="h-4 w-4 text-muted-foreground" />
              <p class="text-xs text-muted-foreground">Status</p>
            </div>
            <div class="flex gap-2 ml-7">
              <Badge>{{ project.status }}</Badge>
              <Badge variant="destructive">{{ project.priority }}</Badge>
            </div>
          </div>
          <Separator />

          <div>
            <div class="flex items-center gap-3 mb-2">
              <IconTag class="h-4 w-4 text-muted-foreground" />
              <p class="text-xs text-muted-foreground">Tags</p>
            </div>
            <div class="flex flex-wrap gap-1 ml-7">
              <Badge v-for="tag in project.tags" :key="tag" variant="secondary" class="text-xs">
                {{ tag }}
              </Badge>
            </div>
          </div>
        </CardContent>
      </Card>
    </div>
  </div>
</template>
```

## Table with Inline Status

For data that fits a tabular layout, use the Table component directly for simple cases or the DataTable composite for richer features like sorting, filtering, and pagination.

```vue
<script setup lang="ts">
import {
  Badge,
  Card,
  CardContent,
  CardHeader,
  CardTitle,
  Table,
  TableBody,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from '@meldui/vue'

const invoices = [
  {
    id: 'INV-001',
    customer: 'Alice Chen',
    status: 'Paid',
    amount: '$250.00',
    date: 'Jan 15, 2025',
  },
  {
    id: 'INV-002',
    customer: 'Bob Smith',
    status: 'Pending',
    amount: '$125.50',
    date: 'Jan 20, 2025',
  },
  {
    id: 'INV-003',
    customer: 'Carol Jones',
    status: 'Overdue',
    amount: '$89.99',
    date: 'Dec 10, 2024',
  },
  { id: 'INV-004', customer: 'David Kim', status: 'Paid', amount: '$340.00', date: 'Jan 22, 2025' },
]

const getStatusVariant = (status: string) => {
  switch (status) {
    case 'Paid':
      return 'default'
    case 'Pending':
      return 'secondary'
    case 'Overdue':
      return 'destructive'
    default:
      return 'outline'
  }
}
</script>

<template>
  <Card>
    <CardHeader>
      <CardTitle>Invoices</CardTitle>
    </CardHeader>
    <CardContent>
      <Table>
        <TableHeader>
          <TableRow>
            <TableHead>Invoice</TableHead>
            <TableHead>Customer</TableHead>
            <TableHead>Status</TableHead>
            <TableHead class="text-right">Amount</TableHead>
            <TableHead>Date</TableHead>
          </TableRow>
        </TableHeader>
        <TableBody>
          <TableRow v-for="invoice in invoices" :key="invoice.id">
            <TableCell class="font-medium">{{ invoice.id }}</TableCell>
            <TableCell>{{ invoice.customer }}</TableCell>
            <TableCell>
              <Badge :variant="getStatusVariant(invoice.status)">{{ invoice.status }}</Badge>
            </TableCell>
            <TableCell class="text-right">{{ invoice.amount }}</TableCell>
            <TableCell class="text-muted-foreground">{{ invoice.date }}</TableCell>
          </TableRow>
        </TableBody>
      </Table>
    </CardContent>
  </Card>
</template>
```

## Empty State

When there is no data to display, use the Empty component to communicate the state clearly and provide a call to action.

```vue
<script setup lang="ts">
import { IconInbox } from '@meldui/tabler-vue'
import { Button, Card, CardContent, Empty } from '@meldui/vue'
</script>

<template>
  <Card>
    <CardContent class="py-12">
      <Empty>
        <template #icon>
          <IconInbox class="h-12 w-12 text-muted-foreground" />
        </template>
        <template #title>No items yet</template>
        <template #description>Get started by creating your first item.</template>
        <template #actions>
          <Button>Create Item</Button>
        </template>
      </Empty>
    </CardContent>
  </Card>
</template>
```

## Design Guidance

**Stat cards.** Use a consistent structure across all stat cards: title at the top, large value, and small change indicator. Color-code trends (green for positive, red for negative) so users can scan quickly.

**List views.** Use Avatar + name/email for people, Badge for status, and a DropdownMenu for row actions. Separate items with `Separator` for clean visual boundaries. Set `p-0` on `CardContent` and apply padding to each row individually so separators span full width.

**Detail views.** Use a two-column grid (`md:grid-cols-3`) with the main content taking two columns and a metadata sidebar taking one. Stack them vertically on mobile. Each metadata field should have a small icon, a label, and the value.

**Empty states.** Always account for the empty case. Use the Empty component with an icon, message, and action button so users know what to do next.

**Loading states.** Replace content with Skeleton components that match the dimensions of the real content. This avoids layout shift and tells users something is loading.
