Form blocks are page-level patterns for collecting user input. They cover three common scenarios: a simple contact form, a structured survey form, and a multi-step wizard form. All patterns use vee-validate with Zod for validation and MeldUI form components for consistent styling.

## Contact Form

A straightforward form with name, email, subject, and message fields. Wrap it in a Card to give it visual boundaries on the page.

```vue
<script setup lang="ts">
import {
  Button,
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
  FormControl,
  FormField,
  FormItem,
  FormLabel,
  FormMessage,
  Input,
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
  Textarea,
} from '@meldui/vue'
import { toTypedSchema } from '@vee-validate/zod'
import { useForm } from 'vee-validate'
import { z } from 'zod'

const contactSchema = z.object({
  name: z.string().min(2, 'Name is required'),
  email: z.string().email('Please enter a valid email'),
  subject: z.enum(['general', 'support', 'billing', 'feedback']),
  message: z.string().min(10, 'Message must be at least 10 characters').max(1000),
})

const { handleSubmit, resetForm } = useForm({
  validationSchema: toTypedSchema(contactSchema),
  initialValues: {
    name: '',
    email: '',
    subject: 'general',
    message: '',
  },
})

const onSubmit = handleSubmit((values) => {
  console.log('Contact form:', values)
  resetForm()
})
</script>

<template>
  <Card class="max-w-2xl mx-auto">
    <CardHeader>
      <CardTitle>Contact Us</CardTitle>
      <CardDescription
        >Fill out the form below and we'll get back to you within 24 hours.</CardDescription
      >
    </CardHeader>
    <CardContent>
      <form class="space-y-4" @submit="onSubmit">
        <div class="grid grid-cols-2 gap-4">
          <FormField v-slot="{ componentField }" name="name">
            <FormItem>
              <FormLabel>Name</FormLabel>
              <FormControl>
                <Input placeholder="Your name" v-bind="componentField" />
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>

          <FormField v-slot="{ componentField }" name="email">
            <FormItem>
              <FormLabel>Email</FormLabel>
              <FormControl>
                <Input type="email" placeholder="you@example.com" v-bind="componentField" />
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>
        </div>

        <FormField v-slot="{ componentField }" name="subject">
          <FormItem>
            <FormLabel>Subject</FormLabel>
            <Select v-bind="componentField">
              <FormControl>
                <SelectTrigger>
                  <SelectValue placeholder="Choose a topic" />
                </SelectTrigger>
              </FormControl>
              <SelectContent>
                <SelectItem value="general">General Inquiry</SelectItem>
                <SelectItem value="support">Technical Support</SelectItem>
                <SelectItem value="billing">Billing</SelectItem>
                <SelectItem value="feedback">Feedback</SelectItem>
              </SelectContent>
            </Select>
            <FormMessage />
          </FormItem>
        </FormField>

        <FormField v-slot="{ componentField }" name="message">
          <FormItem>
            <FormLabel>Message</FormLabel>
            <FormControl>
              <Textarea placeholder="How can we help?" :rows="5" v-bind="componentField" />
            </FormControl>
            <FormMessage />
          </FormItem>
        </FormField>

        <div class="flex justify-end gap-2">
          <Button type="button" variant="outline" @click="resetForm()">Clear</Button>
          <Button type="submit">Send Message</Button>
        </div>
      </form>
    </CardContent>
  </Card>
</template>
```

## Survey Form

A survey form collects structured feedback using radio groups, checkboxes, sliders, and text areas. Group related questions inside Cards with clear headings.

```vue
<script setup lang="ts">
import {
  Button,
  Card,
  CardContent,
  CardDescription,
  CardHeader,
  CardTitle,
  Checkbox,
  FormControl,
  FormDescription,
  FormField,
  FormItem,
  FormLabel,
  FormMessage,
  RadioGroup,
  RadioGroupItem,
  Separator,
  Slider,
  Textarea,
} from '@meldui/vue'
import { toTypedSchema } from '@vee-validate/zod'
import { useForm } from 'vee-validate'
import { z } from 'zod'

const surveySchema = z.object({
  satisfaction: z.enum([
    'very-satisfied',
    'satisfied',
    'neutral',
    'dissatisfied',
    'very-dissatisfied',
  ]),
  recommend: z.number().min(0).max(10),
  features: z.array(z.string()).min(1, 'Select at least one feature'),
  comments: z.string().optional(),
})

const { handleSubmit, values, setFieldValue } = useForm({
  validationSchema: toTypedSchema(surveySchema),
  initialValues: {
    satisfaction: undefined,
    recommend: 5,
    features: [],
    comments: '',
  },
})

const featureOptions = [
  { value: 'components', label: 'Component library' },
  { value: 'theming', label: 'Theming system' },
  { value: 'icons', label: 'Icon packages' },
  { value: 'docs', label: 'Documentation' },
  { value: 'data-table', label: 'DataTable' },
]

const toggleFeature = (feature: string) => {
  const current = values.features || []
  const updated = current.includes(feature)
    ? current.filter((f: string) => f !== feature)
    : [...current, feature]
  setFieldValue('features', updated)
}

const onSubmit = handleSubmit((vals) => {
  console.log('Survey:', vals)
})
</script>

<template>
  <div class="max-w-2xl mx-auto space-y-6">
    <div>
      <h1 class="text-2xl font-bold">Product Feedback Survey</h1>
      <p class="text-muted-foreground">Help us improve by answering a few questions.</p>
    </div>

    <form class="space-y-6" @submit="onSubmit">
      <!-- Overall Satisfaction -->
      <Card>
        <CardHeader>
          <CardTitle class="text-lg">Overall Satisfaction</CardTitle>
          <CardDescription>How satisfied are you with the product?</CardDescription>
        </CardHeader>
        <CardContent>
          <FormField v-slot="{ componentField }" name="satisfaction">
            <FormItem>
              <FormControl>
                <RadioGroup v-bind="componentField" class="space-y-2">
                  <div
                    v-for="option in [
                      { value: 'very-satisfied', label: 'Very Satisfied' },
                      { value: 'satisfied', label: 'Satisfied' },
                      { value: 'neutral', label: 'Neutral' },
                      { value: 'dissatisfied', label: 'Dissatisfied' },
                      { value: 'very-dissatisfied', label: 'Very Dissatisfied' },
                    ]"
                    :key="option.value"
                    class="flex items-center gap-2"
                  >
                    <RadioGroupItem :value="option.value" :id="option.value" />
                    <FormLabel :for="option.value" class="font-normal">{{
                      option.label
                    }}</FormLabel>
                  </div>
                </RadioGroup>
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>
        </CardContent>
      </Card>

      <!-- NPS Score -->
      <Card>
        <CardHeader>
          <CardTitle class="text-lg">Recommendation Score</CardTitle>
          <CardDescription>How likely are you to recommend us? (0-10)</CardDescription>
        </CardHeader>
        <CardContent>
          <FormField name="recommend">
            <FormItem>
              <FormControl>
                <div class="space-y-4">
                  <Slider
                    :model-value="[values.recommend]"
                    :min="0"
                    :max="10"
                    :step="1"
                    @update:model-value="(v) => setFieldValue('recommend', v[0])"
                  />
                  <div class="flex justify-between text-xs text-muted-foreground">
                    <span>Not likely (0)</span>
                    <span class="font-medium text-foreground text-sm">{{ values.recommend }}</span>
                    <span>Very likely (10)</span>
                  </div>
                </div>
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>
        </CardContent>
      </Card>

      <!-- Feature Selection -->
      <Card>
        <CardHeader>
          <CardTitle class="text-lg">Favorite Features</CardTitle>
          <CardDescription>Select the features you use most</CardDescription>
        </CardHeader>
        <CardContent>
          <FormField name="features">
            <FormItem>
              <div class="space-y-3">
                <div
                  v-for="option in featureOptions"
                  :key="option.value"
                  class="flex items-center gap-2"
                >
                  <Checkbox
                    :checked="values.features?.includes(option.value)"
                    @update:checked="() => toggleFeature(option.value)"
                  />
                  <label class="text-sm">{{ option.label }}</label>
                </div>
              </div>
              <FormMessage />
            </FormItem>
          </FormField>
        </CardContent>
      </Card>

      <!-- Comments -->
      <Card>
        <CardHeader>
          <CardTitle class="text-lg">Additional Comments</CardTitle>
        </CardHeader>
        <CardContent>
          <FormField v-slot="{ componentField }" name="comments">
            <FormItem>
              <FormControl>
                <Textarea
                  placeholder="Anything else you'd like to share?"
                  :rows="4"
                  v-bind="componentField"
                />
              </FormControl>
              <FormDescription>Optional</FormDescription>
              <FormMessage />
            </FormItem>
          </FormField>
        </CardContent>
      </Card>

      <Button type="submit" class="w-full">Submit Survey</Button>
    </form>
  </div>
</template>
```

## Multi-Step Form

A wizard-style form that breaks a long process into manageable steps. Use the Stepper component to show progress and navigate between steps.

```vue
<script setup lang="ts">
import {
  Button,
  Card,
  CardContent,
  CardFooter,
  CardHeader,
  CardTitle,
  FormControl,
  FormDescription,
  FormField,
  FormItem,
  FormLabel,
  FormMessage,
  Input,
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
  Separator,
  Stepper,
  StepperDescription,
  StepperIndicator,
  StepperItem,
  StepperSeparator,
  StepperTitle,
  StepperTrigger,
  Textarea,
} from '@meldui/vue'
import { toTypedSchema } from '@vee-validate/zod'
import { useForm } from 'vee-validate'
import { ref } from 'vue'
import { z } from 'zod'

const currentStep = ref(1)

// Each step has its own schema for independent validation
const personalSchema = z.object({
  firstName: z.string().min(1, 'First name is required'),
  lastName: z.string().min(1, 'Last name is required'),
  email: z.string().email('Invalid email address'),
})

const companySchema = z.object({
  company: z.string().min(1, 'Company name is required'),
  role: z.string().min(1, 'Role is required'),
  size: z.enum(['1-10', '11-50', '51-200', '201-1000', '1000+']),
})

const detailsSchema = z.object({
  useCase: z.string().min(20, 'Please describe your use case in at least 20 characters'),
})

const fullSchema = personalSchema.merge(companySchema).merge(detailsSchema)

const { handleSubmit, values, validate } = useForm({
  validationSchema: toTypedSchema(fullSchema),
  initialValues: {
    firstName: '',
    lastName: '',
    email: '',
    company: '',
    role: '',
    size: '1-10',
    useCase: '',
  },
})

const nextStep = async () => {
  // Validate only the fields for the current step
  const result = await validate()
  const fieldsForStep: Record<number, string[]> = {
    1: ['firstName', 'lastName', 'email'],
    2: ['company', 'role', 'size'],
    3: ['useCase'],
  }
  const stepFields = fieldsForStep[currentStep.value] || []
  const hasErrors = stepFields.some((field) => result.errors[field])

  if (!hasErrors) {
    currentStep.value++
  }
}

const prevStep = () => {
  currentStep.value--
}

const onSubmit = handleSubmit((vals) => {
  console.log('Multi-step form:', vals)
})
</script>

<template>
  <Card class="max-w-2xl mx-auto">
    <CardHeader>
      <CardTitle>Get Started</CardTitle>
      <Stepper v-model="currentStep" class="mt-4">
        <StepperItem :step="1">
          <StepperTrigger>
            <StepperIndicator />
            <div class="flex flex-col">
              <StepperTitle>Personal</StepperTitle>
              <StepperDescription>Your information</StepperDescription>
            </div>
          </StepperTrigger>
          <StepperSeparator />
        </StepperItem>
        <StepperItem :step="2">
          <StepperTrigger>
            <StepperIndicator />
            <div class="flex flex-col">
              <StepperTitle>Company</StepperTitle>
              <StepperDescription>Your organization</StepperDescription>
            </div>
          </StepperTrigger>
          <StepperSeparator />
        </StepperItem>
        <StepperItem :step="3">
          <StepperTrigger>
            <StepperIndicator />
            <div class="flex flex-col">
              <StepperTitle>Details</StepperTitle>
              <StepperDescription>Use case</StepperDescription>
            </div>
          </StepperTrigger>
        </StepperItem>
      </Stepper>
    </CardHeader>

    <Separator />

    <form @submit="onSubmit">
      <CardContent class="pt-6">
        <!-- Step 1: Personal Info -->
        <div v-show="currentStep === 1" class="space-y-4">
          <div class="grid grid-cols-2 gap-4">
            <FormField v-slot="{ componentField }" name="firstName">
              <FormItem>
                <FormLabel>First Name</FormLabel>
                <FormControl>
                  <Input v-bind="componentField" placeholder="Jane" />
                </FormControl>
                <FormMessage />
              </FormItem>
            </FormField>

            <FormField v-slot="{ componentField }" name="lastName">
              <FormItem>
                <FormLabel>Last Name</FormLabel>
                <FormControl>
                  <Input v-bind="componentField" placeholder="Doe" />
                </FormControl>
                <FormMessage />
              </FormItem>
            </FormField>
          </div>

          <FormField v-slot="{ componentField }" name="email">
            <FormItem>
              <FormLabel>Email</FormLabel>
              <FormControl>
                <Input type="email" v-bind="componentField" placeholder="jane@example.com" />
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>
        </div>

        <!-- Step 2: Company Info -->
        <div v-show="currentStep === 2" class="space-y-4">
          <FormField v-slot="{ componentField }" name="company">
            <FormItem>
              <FormLabel>Company</FormLabel>
              <FormControl>
                <Input v-bind="componentField" placeholder="Acme Inc." />
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>

          <FormField v-slot="{ componentField }" name="role">
            <FormItem>
              <FormLabel>Role</FormLabel>
              <FormControl>
                <Input v-bind="componentField" placeholder="Frontend Developer" />
              </FormControl>
              <FormMessage />
            </FormItem>
          </FormField>

          <FormField v-slot="{ componentField }" name="size">
            <FormItem>
              <FormLabel>Company Size</FormLabel>
              <Select v-bind="componentField">
                <FormControl>
                  <SelectTrigger>
                    <SelectValue placeholder="Select size" />
                  </SelectTrigger>
                </FormControl>
                <SelectContent>
                  <SelectItem value="1-10">1-10 employees</SelectItem>
                  <SelectItem value="11-50">11-50 employees</SelectItem>
                  <SelectItem value="51-200">51-200 employees</SelectItem>
                  <SelectItem value="201-1000">201-1,000 employees</SelectItem>
                  <SelectItem value="1000+">1,000+ employees</SelectItem>
                </SelectContent>
              </Select>
              <FormMessage />
            </FormItem>
          </FormField>
        </div>

        <!-- Step 3: Use Case -->
        <div v-show="currentStep === 3" class="space-y-4">
          <FormField v-slot="{ componentField }" name="useCase">
            <FormItem>
              <FormLabel>How will you use MeldUI?</FormLabel>
              <FormControl>
                <Textarea
                  v-bind="componentField"
                  placeholder="Describe your project and how MeldUI will help..."
                  :rows="6"
                />
              </FormControl>
              <FormDescription>At least 20 characters</FormDescription>
              <FormMessage />
            </FormItem>
          </FormField>
        </div>
      </CardContent>

      <Separator />

      <CardFooter class="flex justify-between pt-6">
        <Button type="button" variant="outline" :disabled="currentStep === 1" @click="prevStep">
          Back
        </Button>
        <Button v-if="currentStep < 3" type="button" @click="nextStep"> Continue </Button>
        <Button v-else type="submit">Submit</Button>
      </CardFooter>
    </form>
  </Card>
</template>
```

## Design Guidance

**Layout.** Use `max-w-2xl mx-auto` for standalone forms so they stay readable and don't stretch too wide. Use `grid grid-cols-2 gap-4` for pairs of short fields (name, city/state) to save vertical space.

**Validation timing.** Validate on submit for simple forms. For multi-step forms, validate each step before allowing the user to proceed. Use `v-show` instead of `v-if` for steps so that form state is preserved when navigating back.

**Multi-step navigation.** The Stepper component provides visual progress. Disable the Back button on the first step and swap the Continue button for a Submit button on the final step.

**Reset and clear.** Provide a Clear or Reset button on contact forms so users can start over. Use `resetForm()` from vee-validate to clear all fields and validation errors at once.

**Grouping.** In survey forms, wrap each question or question group in its own Card. This gives clear visual separation and makes long forms feel less overwhelming.
