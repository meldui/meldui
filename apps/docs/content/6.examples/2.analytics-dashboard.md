This example walks through building an analytics dashboard using MeldUI's chart system, DataTable composite, and layout components. The dashboard displays key metrics, visualizes trends over time, and provides a filterable data table for detailed exploration.

## Page Structure

An analytics dashboard typically has four layers stacked vertically: a header with filters, a stats row, a chart section, and a data table. Wrap the entire page in `space-y-6` for consistent spacing.

```vue
<script setup lang="ts">
import { IconDownload } from '@meldui/tabler-vue'
import {
  Button,
  DateRangePicker,
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from '@meldui/vue'
import { ref } from 'vue'

import ChartsSection from './ChartsSection.vue'
import DataSection from './DataSection.vue'
import StatsRow from './StatsRow.vue'

const dateRange = ref({ start: '2025-01-01', end: '2025-03-31' })
const channel = ref('all')
</script>

<template>
  <div class="space-y-6">
    <!-- Header with Filters -->
    <div class="flex flex-col gap-4 md:flex-row md:items-center md:justify-between">
      <div>
        <h1 class="text-3xl font-bold tracking-tight">Analytics</h1>
        <p class="text-muted-foreground">Track performance across your channels</p>
      </div>
      <div class="flex items-center gap-3">
        <DateRangePicker v-model="dateRange" />
        <Select v-model="channel">
          <SelectTrigger class="w-[160px]">
            <SelectValue placeholder="Channel" />
          </SelectTrigger>
          <SelectContent>
            <SelectItem value="all">All Channels</SelectItem>
            <SelectItem value="organic">Organic</SelectItem>
            <SelectItem value="paid">Paid</SelectItem>
            <SelectItem value="referral">Referral</SelectItem>
          </SelectContent>
        </Select>
        <Button variant="outline">
          <IconDownload class="mr-2 h-4 w-4" />
          Export
        </Button>
      </div>
    </div>

    <!-- Content Sections -->
    <StatsRow />
    <ChartsSection />
    <DataSection />
  </div>
</template>
```

## Stats Row

Display the primary KPIs in a four-column grid. Each stat card shows the metric name, current value, and a percentage change from the previous period.

```vue
<script setup lang="ts">
import {
  IconArrowDownRight,
  IconArrowUpRight,
  IconCash,
  IconClick,
  IconEye,
  IconUsers,
} from '@meldui/tabler-vue'
import { Card, CardContent, CardHeader, CardTitle } from '@meldui/vue'

const metrics = [
  { title: 'Page Views', value: '573,281', change: '+12.5%', positive: true, icon: IconEye },
  { title: 'Unique Visitors', value: '48,920', change: '+8.2%', positive: true, icon: IconUsers },
  { title: 'Conversion Rate', value: '3.24%', change: '-0.4%', positive: false, icon: IconClick },
  { title: 'Revenue', value: '$128,430', change: '+22.1%', positive: true, icon: IconCash },
]
</script>

<template>
  <div class="grid gap-4 md:grid-cols-2 lg:grid-cols-4">
    <Card v-for="metric in metrics" :key="metric.title">
      <CardHeader class="flex flex-row items-center justify-between space-y-0 pb-2">
        <CardTitle class="text-sm font-medium">{{ metric.title }}</CardTitle>
        <component :is="metric.icon" class="h-4 w-4 text-muted-foreground" />
      </CardHeader>
      <CardContent>
        <div class="text-2xl font-bold">{{ metric.value }}</div>
        <div class="flex items-center text-xs mt-1">
          <component
            :is="metric.positive ? IconArrowUpRight : IconArrowDownRight"
            class="h-3 w-3 mr-1"
            :class="metric.positive ? 'text-green-600' : 'text-red-600'"
          />
          <span :class="metric.positive ? 'text-green-600' : 'text-red-600'">
            {{ metric.change }}
          </span>
          <span class="text-muted-foreground ml-1">vs. previous period</span>
        </div>
      </CardContent>
    </Card>
  </div>
</template>
```

## Charts Section

Use the MeldUI chart system, which wraps the Unovis charting library, to render line and bar charts. The `ChartContainer` component manages the chart config (colors, labels) while you use Unovis components for the actual visualization.

### Line Chart for Trends

A line chart showing page views over time. Define a `ChartConfig` to map data series to labels and colors, then wrap Unovis components inside `ChartContainer`.

```vue
<script setup lang="ts">
import {
  type ChartConfig,
  ChartContainer,
  ChartCrosshair,
  ChartLegendContent,
  ChartTooltip,
  ChartTooltipContent,
} from '@meldui/vue'
import { VisAxis, VisLine, VisXYContainer } from '@unovis/vue'
import { ref } from 'vue'

const chartConfig: ChartConfig = {
  views: { label: 'Page Views', color: 'hsl(var(--chart-1))' },
  visitors: { label: 'Unique Visitors', color: 'hsl(var(--chart-2))' },
}

const data = ref([
  { date: 'Jan', views: 18600, visitors: 8200 },
  { date: 'Feb', views: 22400, visitors: 9800 },
  { date: 'Mar', views: 28100, visitors: 12400 },
  { date: 'Apr', views: 24800, visitors: 11200 },
  { date: 'May', views: 31200, visitors: 14600 },
  { date: 'Jun', views: 35800, visitors: 16800 },
])

const x = (_: unknown, i: number) => i
const yViews = (d: { views: number }) => d.views
const yVisitors = (d: { visitors: number }) => d.visitors
</script>

<template>
  <ChartContainer :config="chartConfig" class="h-[300px] w-full">
    <VisXYContainer :data="data" :margin="{ top: 10, right: 10, bottom: 30, left: 50 }">
      <VisLine :x="x" :y="yViews" color="hsl(var(--chart-1))" />
      <VisLine :x="x" :y="yVisitors" color="hsl(var(--chart-2))" />
      <VisAxis type="x" :tick-format="(i: number) => data[i]?.date ?? ''" />
      <VisAxis type="y" />
      <ChartCrosshair :template="(d: unknown) => ''" />
      <ChartTooltip>
        <ChartTooltipContent />
      </ChartTooltip>
    </VisXYContainer>
    <ChartLegendContent />
  </ChartContainer>
</template>
```

### Bar Chart for Breakdown

A bar chart showing traffic by channel. This complements the line chart and gives users a different perspective on the same data.

```vue
<script setup lang="ts">
import { type ChartConfig, ChartContainer } from '@meldui/vue'
import { Card, CardContent, CardDescription, CardHeader, CardTitle } from '@meldui/vue'
import { VisAxis, VisStackedBar, VisXYContainer } from '@unovis/vue'

const chartConfig: ChartConfig = {
  organic: { label: 'Organic', color: 'hsl(var(--chart-1))' },
  paid: { label: 'Paid', color: 'hsl(var(--chart-2))' },
  referral: { label: 'Referral', color: 'hsl(var(--chart-3))' },
}

const data = [
  { month: 'Jan', organic: 12400, paid: 4200, referral: 2000 },
  { month: 'Feb', organic: 14800, paid: 5100, referral: 2500 },
  { month: 'Mar', organic: 18200, paid: 6400, referral: 3500 },
  { month: 'Apr', organic: 16100, paid: 5800, referral: 2900 },
  { month: 'May', organic: 20400, paid: 7200, referral: 3600 },
  { month: 'Jun', organic: 23600, paid: 8100, referral: 4100 },
]

const x = (_: unknown, i: number) => i
const y = [
  (d: { organic: number }) => d.organic,
  (d: { paid: number }) => d.paid,
  (d: { referral: number }) => d.referral,
]
</script>

<template>
  <Card>
    <CardHeader>
      <CardTitle>Traffic by Channel</CardTitle>
      <CardDescription>Monthly breakdown of traffic sources</CardDescription>
    </CardHeader>
    <CardContent>
      <ChartContainer :config="chartConfig" class="h-[300px] w-full">
        <VisXYContainer :data="data" :margin="{ top: 10, right: 10, bottom: 30, left: 50 }">
          <VisStackedBar :x="x" :y="y" />
          <VisAxis type="x" :tick-format="(i: number) => data[i]?.month ?? ''" />
          <VisAxis type="y" />
        </VisXYContainer>
      </ChartContainer>
    </CardContent>
  </Card>
</template>
```

## Data Table with Filters

Below the charts, provide a detailed data table using the DataTable composite. This gives users the ability to sort, filter, and paginate through individual records.

```vue
<script setup lang="ts">
import {
  Badge,
  Card,
  CardContent,
  CardHeader,
  CardTitle,
  DataTable,
  DataTablePagination,
  DataTableToolbar,
  createColumnHelper,
  useDataTable,
} from '@meldui/vue'
import { h } from 'vue'

interface PageRecord {
  path: string
  title: string
  views: number
  uniqueVisitors: number
  bounceRate: number
  avgDuration: string
  source: 'organic' | 'paid' | 'referral' | 'direct'
}

const data: PageRecord[] = [
  {
    path: '/',
    title: 'Homepage',
    views: 45200,
    uniqueVisitors: 32100,
    bounceRate: 35.2,
    avgDuration: '2m 15s',
    source: 'organic',
  },
  {
    path: '/pricing',
    title: 'Pricing',
    views: 18400,
    uniqueVisitors: 15800,
    bounceRate: 28.1,
    avgDuration: '3m 42s',
    source: 'paid',
  },
  {
    path: '/docs',
    title: 'Documentation',
    views: 32100,
    uniqueVisitors: 24600,
    bounceRate: 18.5,
    avgDuration: '5m 10s',
    source: 'organic',
  },
  {
    path: '/blog',
    title: 'Blog',
    views: 12800,
    uniqueVisitors: 10200,
    bounceRate: 42.3,
    avgDuration: '1m 48s',
    source: 'referral',
  },
  {
    path: '/about',
    title: 'About',
    views: 8600,
    uniqueVisitors: 7100,
    bounceRate: 55.0,
    avgDuration: '0m 52s',
    source: 'direct',
  },
]

const col = createColumnHelper<PageRecord>()

const columns = [
  col.accessor('title', { header: 'Page' }),
  col.accessor('path', { header: 'Path' }),
  col.accessor('views', {
    header: 'Views',
    cell: ({ row }) => row.original.views.toLocaleString(),
  }),
  col.accessor('uniqueVisitors', {
    header: 'Unique Visitors',
    cell: ({ row }) => row.original.uniqueVisitors.toLocaleString(),
  }),
  col.accessor('bounceRate', {
    header: 'Bounce Rate',
    cell: ({ row }) => `${row.original.bounceRate}%`,
  }),
  col.accessor('avgDuration', { header: 'Avg. Duration' }),
  col.accessor('source', {
    header: 'Source',
    cell: ({ row }) => {
      const variants: Record<string, string> = {
        organic: 'default',
        paid: 'secondary',
        referral: 'outline',
        direct: 'outline',
      }
      return h(
        Badge,
        { variant: variants[row.original.source] || 'outline' },
        () => row.original.source,
      )
    },
  }),
]

const filterFields = [
  { id: 'title', label: 'Page', type: 'text' as const },
  {
    id: 'source',
    label: 'Source',
    type: 'select' as const,
    options: [
      { label: 'Organic', value: 'organic' },
      { label: 'Paid', value: 'paid' },
      { label: 'Referral', value: 'referral' },
      { label: 'Direct', value: 'direct' },
    ],
  },
]

const { table } = useDataTable({
  data,
  columns,
  filterFields,
})
</script>

<template>
  <Card>
    <CardHeader>
      <CardTitle>Page Performance</CardTitle>
    </CardHeader>
    <CardContent class="space-y-4">
      <DataTableToolbar :table="table" :filter-fields="filterFields" />
      <DataTable :table="table" />
      <DataTablePagination :table="table" />
    </CardContent>
  </Card>
</template>
```

## Design Guidance

**Filter placement.** Place global filters (date range, channel) in the page header so they affect all sections. Place table-specific filters (text search, column filters) inside the DataTable toolbar.

**Chart pairing.** Use a line chart for trends over time and a bar chart for categorical breakdowns side by side. This gives users both a temporal and compositional view of the same data.

**Consistent color mapping.** Define chart colors using CSS custom properties (`--chart-1`, `--chart-2`, etc.) through `ChartConfig`. This ensures colors match between charts and adapt to light/dark themes automatically.

**Progressive disclosure.** Show summary stats at the top for quick scanning, charts in the middle for visual trends, and a detailed data table at the bottom for drill-down. Users who need more detail scroll further.

**Export.** Provide an export button in the header for downloading the current view as CSV or PDF. This is especially valuable for analytics dashboards where users need to share data with stakeholders.
