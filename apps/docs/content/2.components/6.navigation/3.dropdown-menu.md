## API Reference

### DropdownMenu

The root container that manages dropdown state.

### DropdownMenuTrigger

The element that toggles the dropdown menu.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### DropdownMenuContent

The floating menu content panel.

| Prop         | Type                           | Default    | Description                                |
| ------------ | ------------------------------ | ---------- | ------------------------------------------ |
| `class`      | `string`                       | —          | Custom CSS classes.                        |
| `align`      | `'start' \| 'center' \| 'end'` | `'center'` | Alignment relative to the trigger.         |
| `sideOffset` | `number`                       | `4`        | Offset from the trigger element in pixels. |

### DropdownMenuItem

A selectable item within the dropdown.

| Prop       | Type      | Default | Description                               |
| ---------- | --------- | ------- | ----------------------------------------- |
| `disabled` | `boolean` | `false` | Whether the item is disabled.             |
| `inset`    | `boolean` | `false` | Add left padding for items without icons. |

### DropdownMenuCheckboxItem

A menu item that acts as a checkbox.

| Prop      | Type      | Default | Description                            |
| --------- | --------- | ------- | -------------------------------------- |
| `checked` | `boolean` | `false` | The checked state. Use with `v-model`. |

### DropdownMenuRadioGroup

Groups radio items together for single selection.

| Prop         | Type     | Default | Description                             |
| ------------ | -------- | ------- | --------------------------------------- |
| `modelValue` | `string` | —       | The selected value. Use with `v-model`. |

### DropdownMenuRadioItem

A menu item that acts as a radio button within a group.

| Prop    | Type     | Default | Description                   |
| ------- | -------- | ------- | ----------------------------- |
| `value` | `string` | —       | The value of this radio item. |

### DropdownMenuLabel

A non-interactive label within the menu.

| Prop    | Type      | Default | Description                                |
| ------- | --------- | ------- | ------------------------------------------ |
| `inset` | `boolean` | `false` | Add left padding to align with menu items. |

### DropdownMenuSeparator

A visual divider between menu sections.

### DropdownMenuGroup

Groups related menu items together.

### DropdownMenuSub

Container for a nested submenu.

### DropdownMenuSubTrigger

The item that opens a nested submenu.

| Prop    | Type      | Default | Description                               |
| ------- | --------- | ------- | ----------------------------------------- |
| `inset` | `boolean` | `false` | Add left padding for items without icons. |

### DropdownMenuSubContent

The floating content of a nested submenu.
