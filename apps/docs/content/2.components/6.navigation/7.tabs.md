## API Reference

### Tabs

The root tabs container.

| Prop           | Type                         | Default        | Description                                    |
| -------------- | ---------------------------- | -------------- | ---------------------------------------------- |
| `defaultValue` | `string`                     | —              | The default active tab when uncontrolled.      |
| `modelValue`   | `string`                     | —              | The controlled active tab. Use with `v-model`. |
| `variant`      | `string`                     | —              | The visual variant of the tabs.                |
| `orientation`  | `'horizontal' \| 'vertical'` | `'horizontal'` | The orientation of the tab list.               |

#### Events

| Event               | Payload  | Description                          |
| ------------------- | -------- | ------------------------------------ |
| `update:modelValue` | `string` | Emitted when the active tab changes. |

### TabsList

Container for tab triggers.

| Prop      | Type                  | Default     | Description                       |
| --------- | --------------------- | ----------- | --------------------------------- |
| `variant` | `'default' \| 'line'` | `'default'` | The visual style of the tab list. |
| `class`   | `string`              | —           | Custom CSS classes.               |

### TabsTrigger

A button that activates its associated tab panel.

| Prop       | Type      | Default | Description                                         |
| ---------- | --------- | ------- | --------------------------------------------------- |
| `value`    | `string`  | —       | The value matching the corresponding `TabsContent`. |
| `disabled` | `boolean` | `false` | Whether this tab trigger is disabled.               |

### TabsContent

The content panel displayed when its associated trigger is active.

| Prop    | Type     | Default | Description                                         |
| ------- | -------- | ------- | --------------------------------------------------- |
| `value` | `string` | —       | The value matching the corresponding `TabsTrigger`. |
| `class` | `string` | —       | Custom CSS classes.                                 |

#### Slots

| Slot      | Props | Description            |
| --------- | ----- | ---------------------- |
| `default` | —     | The tab panel content. |
