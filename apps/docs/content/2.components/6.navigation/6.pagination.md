## API Reference

### Pagination

The root pagination container.

| Prop           | Type      | Default | Description                                                    |
| -------------- | --------- | ------- | -------------------------------------------------------------- |
| `total`        | `number`  | —       | Total number of items to paginate.                             |
| `defaultPage`  | `number`  | `1`     | The default active page when uncontrolled.                     |
| `page`         | `number`  | —       | The controlled active page. Use with `v-model`.                |
| `itemsPerPage` | `number`  | `10`    | Number of items displayed per page.                            |
| `siblingCount` | `number`  | `1`     | Number of page buttons shown on each side of the current page. |
| `showEdges`    | `boolean` | `false` | Whether to always show first and last page buttons.            |

#### Events

| Event         | Payload  | Description                           |
| ------------- | -------- | ------------------------------------- |
| `update:page` | `number` | Emitted when the active page changes. |

### PaginationContent

Container for pagination items, provides the `items` slot prop for rendering.

#### Slots

| Slot      | Props       | Description                                                                                      |
| --------- | ----------- | ------------------------------------------------------------------------------------------------ |
| `default` | `{ items }` | The list of page items to render. Each item has a `type` (`'page'` or `'ellipsis'`) and `value`. |

### PaginationItem

Wrapper for individual page number buttons.

| Prop    | Type     | Default | Description                           |
| ------- | -------- | ------- | ------------------------------------- |
| `value` | `number` | —       | The page number this item represents. |

### PaginationFirst

Button to navigate to the first page.

### PaginationPrevious

Button to navigate to the previous page.

### PaginationNext

Button to navigate to the next page.

### PaginationLast

Button to navigate to the last page.

### PaginationEllipsis

Indicator for skipped page numbers.

| Prop    | Type     | Default | Description                         |
| ------- | -------- | ------- | ----------------------------------- |
| `index` | `number` | —       | The position index of the ellipsis. |
