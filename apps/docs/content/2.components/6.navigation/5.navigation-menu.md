## API Reference

### NavigationMenu

The root navigation container.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### NavigationMenuList

The list container for navigation items.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### NavigationMenuItem

A single item within the navigation menu.

### NavigationMenuTrigger

The button that toggles a dropdown content panel.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### NavigationMenuContent

The dropdown content panel displayed when a trigger is activated.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description                                  |
| --------- | ----- | -------------------------------------------- |
| `default` | —     | The content displayed in the dropdown panel. |

### NavigationMenuLink

A navigation link within the menu.

| Prop     | Type      | Default | Description                                   |
| -------- | --------- | ------- | --------------------------------------------- |
| `href`   | `string`  | —       | The URL the link points to.                   |
| `active` | `boolean` | `false` | Whether the link represents the current page. |

### NavigationMenuViewport

The animated viewport container for dropdown content.

### NavigationMenuIndicator

An animated indicator that highlights the active trigger.
