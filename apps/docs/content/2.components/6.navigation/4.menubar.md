## API Reference

### Menubar

The root horizontal menu bar container.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### MenubarMenu

A single menu within the menu bar.

### MenubarTrigger

The button that opens a menu's dropdown content.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### MenubarContent

The floating dropdown content of a menu.

| Prop         | Type                           | Default   | Description                        |
| ------------ | ------------------------------ | --------- | ---------------------------------- |
| `align`      | `'start' \| 'center' \| 'end'` | `'start'` | Alignment relative to the trigger. |
| `sideOffset` | `number`                       | `8`       | Offset from the trigger in pixels. |

### MenubarItem

A selectable item within a menu.

| Prop       | Type      | Default | Description                               |
| ---------- | --------- | ------- | ----------------------------------------- |
| `disabled` | `boolean` | `false` | Whether the item is disabled.             |
| `inset`    | `boolean` | `false` | Add left padding for items without icons. |

### MenubarSeparator

A visual divider between menu items.

### MenubarLabel

A non-interactive label within a menu.

| Prop    | Type      | Default | Description                                |
| ------- | --------- | ------- | ------------------------------------------ |
| `inset` | `boolean` | `false` | Add left padding to align with menu items. |

### MenubarCheckboxItem

A menu item that acts as a checkbox.

| Prop      | Type      | Default | Description                            |
| --------- | --------- | ------- | -------------------------------------- |
| `checked` | `boolean` | `false` | The checked state. Use with `v-model`. |

### MenubarRadioGroup

Groups radio items together for single selection.

| Prop         | Type     | Default | Description                             |
| ------------ | -------- | ------- | --------------------------------------- |
| `modelValue` | `string` | —       | The selected value. Use with `v-model`. |

### MenubarRadioItem

A menu item that acts as a radio button within a group.

| Prop    | Type     | Default | Description                   |
| ------- | -------- | ------- | ----------------------------- |
| `value` | `string` | —       | The value of this radio item. |

### MenubarSub

Container for a nested submenu.

### MenubarSubTrigger

The item that opens a nested submenu.

| Prop    | Type      | Default | Description                               |
| ------- | --------- | ------- | ----------------------------------------- |
| `inset` | `boolean` | `false` | Add left padding for items without icons. |

### MenubarSubContent

The floating content of a nested submenu.
