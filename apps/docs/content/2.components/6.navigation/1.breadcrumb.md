## API Reference

### Breadcrumb

The root navigation container with `aria-label="breadcrumb"`.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### BreadcrumbList

Ordered list wrapper for breadcrumb items.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### BreadcrumbItem

Individual item within the breadcrumb trail.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### BreadcrumbLink

A navigable link within a breadcrumb item.

| Prop      | Type      | Default | Description                                     |
| --------- | --------- | ------- | ----------------------------------------------- |
| `href`    | `string`  | —       | The URL the link points to.                     |
| `asChild` | `boolean` | `false` | Render as a child element instead of an anchor. |

### BreadcrumbPage

The current page indicator (not a link).

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### BreadcrumbSeparator

Visual separator between breadcrumb items (defaults to `/`).

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description                                  |
| --------- | ----- | -------------------------------------------- |
| `default` | —     | Custom separator content (icon, text, etc.). |

### BreadcrumbEllipsis

Truncation indicator for long breadcrumb trails.
