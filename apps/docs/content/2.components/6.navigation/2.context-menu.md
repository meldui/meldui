## API Reference

### ContextMenu

The root container that manages context menu state.

### ContextMenuTrigger

The element that triggers the context menu on right-click.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### ContextMenuContent

The floating menu content panel.

| Prop          | Type     | Default | Description                      |
| ------------- | -------- | ------- | -------------------------------- |
| `class`       | `string` | —       | Custom CSS classes.              |
| `alignOffset` | `number` | —       | Offset from the alignment edge.  |
| `sideOffset`  | `number` | —       | Offset from the trigger element. |

### ContextMenuItem

A selectable item within the context menu.

| Prop       | Type      | Default | Description                               |
| ---------- | --------- | ------- | ----------------------------------------- |
| `disabled` | `boolean` | `false` | Whether the item is disabled.             |
| `inset`    | `boolean` | `false` | Add left padding for items without icons. |

### ContextMenuCheckboxItem

A menu item that acts as a checkbox.

| Prop      | Type      | Default | Description                            |
| --------- | --------- | ------- | -------------------------------------- |
| `checked` | `boolean` | `false` | The checked state. Use with `v-model`. |

### ContextMenuRadioGroup

Groups radio items together for single selection.

| Prop         | Type     | Default | Description                             |
| ------------ | -------- | ------- | --------------------------------------- |
| `modelValue` | `string` | —       | The selected value. Use with `v-model`. |

### ContextMenuRadioItem

A menu item that acts as a radio button within a group.

| Prop    | Type     | Default | Description                   |
| ------- | -------- | ------- | ----------------------------- |
| `value` | `string` | —       | The value of this radio item. |

### ContextMenuLabel

A non-interactive label within the menu.

| Prop    | Type      | Default | Description                                |
| ------- | --------- | ------- | ------------------------------------------ |
| `inset` | `boolean` | `false` | Add left padding to align with menu items. |

### ContextMenuSeparator

A visual divider between menu sections.

### ContextMenuSub

Container for a nested submenu.

### ContextMenuSubTrigger

The item that opens a nested submenu on hover.

| Prop    | Type      | Default | Description                               |
| ------- | --------- | ------- | ----------------------------------------- |
| `inset` | `boolean` | `false` | Add left padding for items without icons. |

### ContextMenuSubContent

The floating content of a nested submenu.
