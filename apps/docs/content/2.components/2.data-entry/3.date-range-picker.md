## API Reference

### Props

| Prop                  | Type                                | Default         | Description                                                  |
| --------------------- | ----------------------------------- | --------------- | ------------------------------------------------------------ |
| `modelValue`          | `DateValue \| DateRange`            | —               | The controlled selected date or range (v-model).             |
| `defaultValue`        | `DateValue \| DateRange`            | —               | The default value for uncontrolled usage.                    |
| `mode`                | `'single' \| 'range'`               | `'range'`       | Selection mode.                                              |
| `placeholder`         | `string`                            | `'Pick a date'` | Placeholder text for the trigger button.                     |
| `presets`             | `DatePreset[] \| DateRangePreset[]` | —               | Custom date presets. Uses built-in defaults if not provided. |
| `showCalendar`        | `boolean`                           | `true`          | Whether to show the calendar in the popover.                 |
| `closeOnPresetSelect` | `boolean`                           | `true`          | Close the popover when a preset is selected.                 |
| `locale`              | `string`                            | `'en'`          | Locale for date formatting.                                  |
| `minValue`            | `DateValue`                         | —               | Minimum selectable date.                                     |
| `maxValue`            | `DateValue`                         | —               | Maximum selectable date.                                     |
| `numberOfMonths`      | `number`                            | —               | Number of months to display.                                 |
| `disabled`            | `boolean`                           | `false`         | Disable the date picker.                                     |
| `readonly`            | `boolean`                           | `false`         | Make the date picker read-only.                              |
| `class`               | `string`                            | —               | Custom CSS classes for the trigger.                          |

### Events

| Event               | Payload                               | Description                             |
| ------------------- | ------------------------------------- | --------------------------------------- |
| `update:modelValue` | `DateValue \| DateRange \| undefined` | Emitted when the selected date changes. |
| `preset-select`     | `DatePreset \| DateRangePreset`       | Emitted when a preset is selected.      |

### Sub-components

For advanced customization, the following sub-components are available:

- `DateRangePickerTrigger` -- The trigger button.
- `DateRangePickerContent` -- The popover content container.
- `DateRangePickerCalendar` -- The calendar view.
- `DateRangePickerPresets` -- The presets list.
