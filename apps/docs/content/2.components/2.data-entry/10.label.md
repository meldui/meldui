## API Reference

### Props

| Prop      | Type      | Default | Description                                                 |
| --------- | --------- | ------- | ----------------------------------------------------------- |
| `for`     | `string`  | —       | The `id` of the form element this label is associated with. |
| `asChild` | `boolean` | `false` | Render as a child element instead of a `<label>`.           |
| `class`   | `string`  | —       | Custom CSS classes.                                         |

### Slots

| Slot      | Description                                            |
| --------- | ------------------------------------------------------ |
| `default` | Label text and optional inline elements (e.g., icons). |
