## API Reference

### Select

The root component that manages selection state.

| Prop           | Type      | Default | Description                               |
| -------------- | --------- | ------- | ----------------------------------------- |
| `modelValue`   | `string`  | —       | The controlled selected value (v-model).  |
| `defaultValue` | `string`  | —       | The default value for uncontrolled usage. |
| `open`         | `boolean` | —       | The controlled open state (v-model:open). |
| `disabled`     | `boolean` | `false` | Disable the select.                       |
| `required`     | `boolean` | `false` | Mark as required for form validation.     |
| `name`         | `string`  | —       | Name attribute for form submission.       |

#### Events

| Event               | Payload   | Description                          |
| ------------------- | --------- | ------------------------------------ |
| `update:modelValue` | `string`  | Emitted when the selection changes.  |
| `update:open`       | `boolean` | Emitted when the open state changes. |

### SelectTrigger

The button that toggles the dropdown.

| Prop    | Type                | Default     | Description         |
| ------- | ------------------- | ----------- | ------------------- |
| `size`  | `'sm' \| 'default'` | `'default'` | The trigger size.   |
| `class` | `string`            | —           | Custom CSS classes. |

### SelectValue

Displays the selected value inside the trigger.

| Prop          | Type     | Default | Description                                 |
| ------------- | -------- | ------- | ------------------------------------------- |
| `placeholder` | `string` | —       | Placeholder text when no value is selected. |

### SelectContent

The dropdown content container.

### SelectGroup

Groups related items with an optional label.

### SelectItem

An individual selectable option.

| Prop       | Type      | Default | Description       |
| ---------- | --------- | ------- | ----------------- |
| `value`    | `string`  | —       | The item value.   |
| `disabled` | `boolean` | `false` | Disable the item. |

### SelectLabel

A non-interactive label for a group of items.

### SelectSeparator

A visual separator between groups.
