## Usage

::ComponentPreview{name="FormBasic"}
::

The `Form` component integrates with [vee-validate](https://vee-validate.logaretm.com/) for schema-based validation.

## API Reference

### Form

The root form element, re-exported from vee-validate.

| Event    | Payload                   | Description                                       |
| -------- | ------------------------- | ------------------------------------------------- |
| `submit` | `Record<string, unknown>` | Emitted with validated values on form submission. |

### FormField

Connects a form field to the validation system. Re-exported from vee-validate.

| Prop   | Type     | Default | Description                                        |
| ------ | -------- | ------- | -------------------------------------------------- |
| `name` | `string` | —       | The field name (must match validation schema key). |

#### Slots

| Slot      | Props                                          | Description                             |
| --------- | ---------------------------------------------- | --------------------------------------- |
| `default` | `{ componentField, value, errorMessage, ... }` | Scoped slot with field binding helpers. |

### FormFieldArray

Manages arrays of fields. Re-exported from vee-validate.

| Prop   | Type     | Default | Description           |
| ------ | -------- | ------- | --------------------- |
| `name` | `string` | —       | The field array name. |

### FormItem

Wrapper that provides context for label, description, and error message.

### FormControl

Wraps the actual input element, connecting it to the form field.

### FormLabel

Accessible label linked to the form control.

| Slot      | Description |
| --------- | ----------- |
| `default` | Label text. |

### FormDescription

Helper text for the form field.

| Slot      | Description       |
| --------- | ----------------- |
| `default` | Description text. |

### FormMessage

Displays the validation error message automatically.
