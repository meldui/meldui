## API Reference

### Props

| Prop             | Type                         | Default | Description                                       |
| ---------------- | ---------------------------- | ------- | ------------------------------------------------- |
| `checked`        | `boolean \| 'indeterminate'` | —       | The controlled checked state (v-model).           |
| `defaultChecked` | `boolean`                    | —       | The default checked state for uncontrolled usage. |
| `disabled`       | `boolean`                    | `false` | Disable the checkbox.                             |
| `required`       | `boolean`                    | `false` | Mark the checkbox as required.                    |
| `name`           | `string`                     | —       | The name attribute for form submission.           |
| `value`          | `string`                     | `'on'`  | The value attribute for form submission.          |
| `id`             | `string`                     | —       | The id attribute.                                 |
| `class`          | `string`                     | —       | Custom CSS classes.                               |

### Events

| Event            | Payload                      | Description                             |
| ---------------- | ---------------------------- | --------------------------------------- |
| `update:checked` | `boolean \| 'indeterminate'` | Emitted when the checked state changes. |

### Slots

| Slot      | Description                                       |
| --------- | ------------------------------------------------- |
| `default` | Custom indicator icon (defaults to a check icon). |
