## API Reference

### PinInput

The root component that manages pin input state.

| Prop           | Type                 | Default  | Description                                                        |
| -------------- | -------------------- | -------- | ------------------------------------------------------------------ |
| `modelValue`   | `string[]`           | —        | The controlled value array (v-model).                              |
| `defaultValue` | `string[]`           | —        | The default value for uncontrolled usage.                          |
| `otp`          | `boolean`            | `true`   | Enable one-time password mode (masks input and uses autocomplete). |
| `type`         | `'text' \| 'number'` | `'text'` | The input type for each slot.                                      |
| `placeholder`  | `string`             | —        | Placeholder character for empty slots.                             |
| `disabled`     | `boolean`            | `false`  | Disable the pin input.                                             |
| `name`         | `string`             | —        | Name attribute for form submission.                                |
| `class`        | `string`             | —        | Custom CSS classes.                                                |

#### Events

| Event               | Payload    | Description                        |
| ------------------- | ---------- | ---------------------------------- |
| `update:modelValue` | `string[]` | Emitted when the value changes.    |
| `complete`          | `string[]` | Emitted when all slots are filled. |

### PinInputGroup

Groups a set of pin input slots together.

### PinInputSlot

An individual character slot.

| Prop    | Type     | Default | Description                  |
| ------- | -------- | ------- | ---------------------------- |
| `index` | `number` | —       | The slot index (zero-based). |

### PinInputSeparator

A visual separator between groups of slots.
