## API Reference

### Field

The root field container.

| Prop          | Type                                         | Default      | Description                    |
| ------------- | -------------------------------------------- | ------------ | ------------------------------ |
| `orientation` | `'vertical' \| 'horizontal' \| 'responsive'` | `'vertical'` | Layout direction of the field. |
| `class`       | `string`                                     | â€”            | Custom CSS classes.            |

### FieldLabel

Accessible label for the field.

| Slot      | Description |
| --------- | ----------- |
| `default` | Label text. |

### FieldContent

Wrapper for the input and supplementary content.

### FieldDescription

Helper text displayed below the input.

| Slot      | Description       |
| --------- | ----------------- |
| `default` | Description text. |

### FieldError

Error message displayed when the field is invalid.

| Slot      | Description         |
| --------- | ------------------- |
| `default` | Error message text. |

### FieldGroup

Groups multiple fields together with `@container` query support.

### FieldSet

Semantic `<fieldset>` wrapper for grouped fields.

### FieldLegend

Legend for a `FieldSet`.

### FieldTitle

Title heading within a `FieldSet`.

### FieldSeparator

Visual separator between fields.
