## API Reference

### InputGroup

The root container that groups an input with addons.

### InputGroupInput

The text input element within the group.

All standard HTML `<input>` attributes are supported.

### InputGroupAddon

Container for addon content such as icons, text, or buttons.

| Prop    | Type                                                             | Default          | Description                                  |
| ------- | ---------------------------------------------------------------- | ---------------- | -------------------------------------------- |
| `align` | `'inline-start' \| 'inline-end' \| 'block-start' \| 'block-end'` | `'inline-start'` | Position of the addon relative to the input. |
| `class` | `string`                                                         | —                | Custom CSS classes.                          |

### InputGroupButton

A button rendered inside the input group.

| Prop      | Type                                                                          | Default | Description                 |
| --------- | ----------------------------------------------------------------------------- | ------- | --------------------------- |
| `variant` | `'default' \| 'secondary' \| 'destructive' \| 'outline' \| 'ghost' \| 'link'` | —       | Visual style of the button. |
| `size`    | `'xs' \| 'sm' \| 'icon-xs' \| 'icon-sm'`                                      | `'xs'`  | Size of the button.         |
| `class`   | `string`                                                                      | —       | Custom CSS classes.         |

### InputGroupText

Static text displayed inside the addon (e.g., `$`, `@`).

### InputGroupTextarea

A textarea element within the group, used instead of `InputGroupInput` for multi-line content.
