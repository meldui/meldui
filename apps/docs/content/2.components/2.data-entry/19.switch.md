## API Reference

### Props

| Prop             | Type                                                                          | Default     | Description                                       |
| ---------------- | ----------------------------------------------------------------------------- | ----------- | ------------------------------------------------- |
| `checked`        | `boolean`                                                                     | —           | The controlled checked state (v-model:checked).   |
| `defaultChecked` | `boolean`                                                                     | —           | The default checked state for uncontrolled usage. |
| `variant`        | `'default' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'neutral'` | `'default'` | Color variant.                                    |
| `disabled`       | `boolean`                                                                     | `false`     | Disable the switch.                               |
| `required`       | `boolean`                                                                     | `false`     | Mark as required for form validation.             |
| `name`           | `string`                                                                      | —           | Name attribute for form submission.               |
| `value`          | `string`                                                                      | `'on'`      | Value attribute for form submission.              |
| `id`             | `string`                                                                      | —           | The id attribute (use with `Label`).              |
| `class`          | `string`                                                                      | —           | Custom CSS classes.                               |

### Events

| Event            | Payload   | Description                             |
| ---------------- | --------- | --------------------------------------- |
| `update:checked` | `boolean` | Emitted when the checked state changes. |

### Slots

| Slot    | Props                  | Description                             |
| ------- | ---------------------- | --------------------------------------- |
| `thumb` | `{ checked: boolean }` | Custom content inside the switch thumb. |
