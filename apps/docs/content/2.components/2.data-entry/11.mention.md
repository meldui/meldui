## API Reference

### Mention

The root component that manages mention state.

| Prop          | Type               | Default           | Description                                                                                    |
| ------------- | ------------------ | ----------------- | ---------------------------------------------------------------------------------------------- |
| `modelValue`  | `string`           | `''`              | The serialized input value with mention markers (v-model).                                     |
| `triggers`    | `MentionTrigger[]` | `[{ char: '@' }]` | Trigger configurations. Each trigger has a `char` (e.g., `@`, `#`) and optional `allowSpaces`. |
| `placeholder` | `string`           | `''`              | Placeholder text.                                                                              |
| `multiline`   | `boolean`          | `false`           | Use a textarea instead of an input.                                                            |
| `disabled`    | `boolean`          | `false`           | Disable all interactions.                                                                      |
| `readonly`    | `boolean`          | `false`           | Read-only mode.                                                                                |
| `loop`        | `boolean`          | `true`            | Enable keyboard navigation looping.                                                            |
| `dir`         | `'ltr' \| 'rtl'`   | `'ltr'`           | Text direction.                                                                                |

#### Events

| Event               | Payload           | Description                                |
| ------------------- | ----------------- | ------------------------------------------ |
| `update:modelValue` | `string`          | Emitted when the serialized value changes. |
| `select`            | `MentionItemData` | Emitted when a mention item is selected.   |
| `remove`            | `MentionData`     | Emitted when a mention is removed.         |

### MentionInput

The text input element.

### MentionContent

The popover container for mention suggestions.

### MentionItem

An individual selectable mention item.

| Prop       | Type      | Default | Description                                       |
| ---------- | --------- | ------- | ------------------------------------------------- |
| `value`    | `string`  | —       | The item value (stored in the serialized output). |
| `label`    | `string`  | —       | The display label.                                |
| `disabled` | `boolean` | `false` | Disable the item.                                 |

### MentionEmpty

Content displayed when no items match the search query.

### MentionLabel

A label for grouping mention items.

### MentionLoading

Loading indicator shown while fetching suggestions.

### MentionHighlighter

Renders highlighted mention tokens in the input overlay.

### MentionPortal

Renders the content in a portal (outside the DOM hierarchy).
