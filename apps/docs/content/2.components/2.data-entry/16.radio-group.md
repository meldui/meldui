## API Reference

### RadioGroup

The root component that manages selection state.

| Prop           | Type                         | Default      | Description                               |
| -------------- | ---------------------------- | ------------ | ----------------------------------------- |
| `modelValue`   | `string`                     | —            | The controlled selected value (v-model).  |
| `defaultValue` | `string`                     | —            | The default value for uncontrolled usage. |
| `disabled`     | `boolean`                    | `false`      | Disable all radio items.                  |
| `required`     | `boolean`                    | `false`      | Mark the group as required.               |
| `name`         | `string`                     | —            | Name attribute for form submission.       |
| `orientation`  | `'horizontal' \| 'vertical'` | `'vertical'` | Layout orientation.                       |
| `loop`         | `boolean`                    | `true`       | Enable keyboard navigation looping.       |
| `class`        | `string`                     | —            | Custom CSS classes.                       |

#### Events

| Event               | Payload  | Description                         |
| ------------------- | -------- | ----------------------------------- |
| `update:modelValue` | `string` | Emitted when the selection changes. |

### RadioGroupItem

An individual radio option.

| Prop       | Type      | Default | Description                          |
| ---------- | --------- | ------- | ------------------------------------ |
| `value`    | `string`  | —       | The item value.                      |
| `disabled` | `boolean` | `false` | Disable this item.                   |
| `id`       | `string`  | —       | The id attribute (use with `Label`). |
