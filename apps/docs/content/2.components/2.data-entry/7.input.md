## API Reference

### Props

| Prop           | Type               | Default | Description                               |
| -------------- | ------------------ | ------- | ----------------------------------------- |
| `modelValue`   | `string \| number` | —       | The controlled input value (v-model).     |
| `defaultValue` | `string \| number` | —       | The default value for uncontrolled usage. |
| `class`        | `string`           | —       | Custom CSS classes.                       |

All standard HTML `<input>` attributes are also supported (e.g., `type`, `placeholder`, `disabled`, `readonly`, `required`).

### Events

| Event               | Payload            | Description                           |
| ------------------- | ------------------ | ------------------------------------- |
| `update:modelValue` | `string \| number` | Emitted when the input value changes. |

### Slots

This component does not have slots.
