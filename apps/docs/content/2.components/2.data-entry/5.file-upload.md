## API Reference

### FileUpload

The root component that manages file state.

| Prop         | Type      | Default    | Description                                                     |
| ------------ | --------- | ---------- | --------------------------------------------------------------- |
| `modelValue` | `File[]`  | `[]`       | The selected files (v-model).                                   |
| `accept`     | `string`  | —          | Accepted file types (MIME types or extensions, e.g. `image/*`). |
| `maxFiles`   | `number`  | `Infinity` | Maximum number of files allowed.                                |
| `maxSize`    | `number`  | —          | Maximum file size in bytes per file.                            |
| `multiple`   | `boolean` | `true`     | Allow multiple file selection.                                  |
| `disabled`   | `boolean` | `false`    | Disable file upload interactions.                               |
| `name`       | `string`  | —          | Name attribute for form integration.                            |
| `class`      | `string`  | —          | Custom CSS classes.                                             |

#### Events

| Event               | Payload             | Description                                        |
| ------------------- | ------------------- | -------------------------------------------------- |
| `update:modelValue` | `File[]`            | Emitted when the file list changes.                |
| `file-accept`       | `File`              | Emitted for each accepted file.                    |
| `file-reject`       | `File, FileError[]` | Emitted for each rejected file with error details. |

### FileUploadDropzone

Drop target area for drag-and-drop uploads.

#### Slots

| Slot      | Props                                                                      | Description       |
| --------- | -------------------------------------------------------------------------- | ----------------- |
| `default` | `{ isDragging: boolean, isDisabled: boolean, openFilePicker: () => void }` | Dropzone content. |

### FileUploadTrigger

Button to open the native file picker.

### FileUploadList

Container for the list of uploaded file items.

### FileUploadItem

Displays an individual file with its metadata.

| Prop       | Type                                                | Default | Description                             |
| ---------- | --------------------------------------------------- | ------- | --------------------------------------- |
| `file`     | `File`                                              | —       | The file this item represents.          |
| `progress` | `number`                                            | —       | Upload progress (0-100).                |
| `status`   | `'pending' \| 'uploading' \| 'complete' \| 'error'` | —       | Current upload status.                  |
| `error`    | `string`                                            | —       | Error message when status is `'error'`. |

### FileUploadItemName

Displays the file name.

### FileUploadItemSize

Displays the formatted file size.

| Prop        | Type     | Default | Description                                   |
| ----------- | -------- | ------- | --------------------------------------------- |
| `precision` | `number` | —       | Number of decimal places for size formatting. |

### FileUploadItemPreview

Displays a thumbnail preview for image files.

### FileUploadItemProgress

Displays a progress bar during upload.

| Prop      | Type                                      | Default     | Description                 |
| --------- | ----------------------------------------- | ----------- | --------------------------- |
| `variant` | `'default' \| 'success' \| 'destructive'` | `'default'` | Progress bar color variant. |

### FileUploadItemDelete

Button to remove a file from the list.

### FileUploadClear

Button to remove all files.
