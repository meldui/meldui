## Autosize

Enable the `autosize` prop to make the textarea grow as the user types. Use `maxHeight` to set a maximum height before scrolling.

::ComponentPreview{name="TextareaAutosize"}
::

## API Reference

### Props

| Prop           | Type               | Default | Description                                                                              |
| -------------- | ------------------ | ------- | ---------------------------------------------------------------------------------------- |
| `modelValue`   | `string \| number` | —       | The controlled value (v-model).                                                          |
| `defaultValue` | `string \| number` | —       | The default value for uncontrolled usage.                                                |
| `autosize`     | `boolean`          | `false` | Automatically resize the textarea to fit its content.                                    |
| `maxHeight`    | `string`           | —       | Maximum height before scrolling when `autosize` is enabled (CSS value, e.g., `'200px'`). |
| `class`        | `string`           | —       | Custom CSS classes.                                                                      |

All standard HTML `<textarea>` attributes are also supported (e.g., `placeholder`, `disabled`, `readonly`, `required`, `rows`).

### Events

| Event               | Payload            | Description                     |
| ------------------- | ------------------ | ------------------------------- |
| `update:modelValue` | `string \| number` | Emitted when the value changes. |

### Slots

This component does not have slots.
