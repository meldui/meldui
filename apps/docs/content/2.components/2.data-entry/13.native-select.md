## API Reference

### NativeSelect

A styled native `<select>` element.

| Prop         | Type     | Default | Description                              |
| ------------ | -------- | ------- | ---------------------------------------- |
| `modelValue` | `string` | —       | The controlled selected value (v-model). |
| `class`      | `string` | —       | Custom CSS classes.                      |

All standard HTML `<select>` attributes are also supported (e.g., `disabled`, `required`, `multiple`, `name`).

#### Events

| Event               | Payload  | Description                         |
| ------------------- | -------- | ----------------------------------- |
| `update:modelValue` | `string` | Emitted when the selection changes. |

### NativeSelectOption

A styled `<option>` element.

| Prop       | Type      | Default | Description         |
| ---------- | --------- | ------- | ------------------- |
| `value`    | `string`  | —       | The option value.   |
| `disabled` | `boolean` | `false` | Disable the option. |

### NativeSelectOptGroup

A styled `<optgroup>` element for grouping options.

| Prop    | Type     | Default | Description      |
| ------- | -------- | ------- | ---------------- |
| `label` | `string` | —       | The group label. |
