## API Reference

### TagsInput

The root component that manages the tags list.

| Prop           | Type       | Default | Description                                     |
| -------------- | ---------- | ------- | ----------------------------------------------- |
| `modelValue`   | `string[]` | —       | The controlled tags array (v-model).            |
| `defaultValue` | `string[]` | —       | The default value for uncontrolled usage.       |
| `addOnPaste`   | `boolean`  | `false` | Add tags from pasted text (split by delimiter). |
| `duplicate`    | `boolean`  | `false` | Allow duplicate tags.                           |
| `disabled`     | `boolean`  | `false` | Disable the tags input.                         |
| `delimiter`    | `string`   | `','`   | Delimiter for splitting pasted text.            |
| `max`          | `number`   | —       | Maximum number of tags allowed.                 |
| `name`         | `string`   | —       | Name attribute for form submission.             |
| `class`        | `string`   | —       | Custom CSS classes.                             |

#### Events

| Event               | Payload    | Description                          |
| ------------------- | ---------- | ------------------------------------ |
| `update:modelValue` | `string[]` | Emitted when the tags array changes. |

### TagsInputInput

The text input for adding new tags.

| Prop          | Type     | Default | Description       |
| ------------- | -------- | ------- | ----------------- |
| `placeholder` | `string` | —       | Placeholder text. |

### TagsInputItem

A single tag item.

| Prop       | Type      | Default | Description       |
| ---------- | --------- | ------- | ----------------- |
| `value`    | `string`  | —       | The tag value.    |
| `disabled` | `boolean` | `false` | Disable this tag. |

### TagsInputItemText

Displays the text content of a tag.

### TagsInputItemDelete

Button to remove a tag.
