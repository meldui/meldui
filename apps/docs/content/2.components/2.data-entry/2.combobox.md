## API Reference

### Combobox

The root component that manages state.

| Prop             | Type                                          | Default | Description                               |
| ---------------- | --------------------------------------------- | ------- | ----------------------------------------- |
| `modelValue`     | `string \| string[]`                          | —       | The controlled selected value (v-model).  |
| `defaultValue`   | `string \| string[]`                          | —       | The default value for uncontrolled usage. |
| `open`           | `boolean`                                     | —       | The controlled open state (v-model:open). |
| `multiple`       | `boolean`                                     | `false` | Allow multiple selections.                |
| `disabled`       | `boolean`                                     | `false` | Disable the combobox.                     |
| `name`           | `string`                                      | —       | The name attribute for form submission.   |
| `filterFunction` | `(items: string[], term: string) => string[]` | —       | Custom filter function.                   |

#### Events

| Event               | Payload              | Description                          |
| ------------------- | -------------------- | ------------------------------------ |
| `update:modelValue` | `string \| string[]` | Emitted when the selection changes.  |
| `update:open`       | `boolean`            | Emitted when the open state changes. |

### ComboboxAnchor

Wraps the trigger and input elements.

### ComboboxInput

The text input for filtering items.

| Prop          | Type     | Default | Description       |
| ------------- | -------- | ------- | ----------------- |
| `placeholder` | `string` | —       | Placeholder text. |

### ComboboxTrigger

Button to toggle the dropdown open/closed.

### ComboboxList

Container for the list of options.

### ComboboxItem

An individual selectable option.

| Prop       | Type      | Default | Description       |
| ---------- | --------- | ------- | ----------------- |
| `value`    | `string`  | —       | The item value.   |
| `disabled` | `boolean` | `false` | Disable the item. |

### ComboboxEmpty

Content displayed when no items match the filter.

### ComboboxGroup

Groups related items together.

### ComboboxItemIndicator

Visual indicator for the selected item.

### ComboboxSeparator

A visual separator between groups.
