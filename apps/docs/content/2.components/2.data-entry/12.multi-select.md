## API Reference

### Props

| Prop                | Type                              | Default | Description                                                                         |
| ------------------- | --------------------------------- | ------- | ----------------------------------------------------------------------------------- |
| `modelValue`        | `string[]`                        | —       | The selected values (v-model).                                                      |
| `options`           | `string[] \| MultiSelectOption[]` | —       | Flat list of options. Each option can be a string or `{ value, label, disabled? }`. |
| `groups`            | `MultiSelectGroup[]`              | —       | Grouped options. Each group has a `label` and `options` array.                      |
| `placeholder`       | `string`                          | —       | Placeholder text for the trigger.                                                   |
| `searchPlaceholder` | `string`                          | —       | Placeholder text for the search input.                                              |
| `emptyText`         | `string`                          | —       | Text shown when no options match the search.                                        |
| `maxDisplay`        | `number`                          | —       | Maximum number of selected items to display in the trigger.                         |
| `max`               | `number`                          | —       | Maximum number of items that can be selected.                                       |
| `disabled`          | `boolean`                         | `false` | Disable the multi-select.                                                           |
| `creatable`         | `boolean`                         | `false` | Allow creating new options from the search input.                                   |
| `createLabel`       | `(value: string) => string`       | —       | Custom label formatter for created options.                                         |

### Events

| Event               | Payload    | Description                         |
| ------------------- | ---------- | ----------------------------------- |
| `update:modelValue` | `string[]` | Emitted when the selection changes. |
