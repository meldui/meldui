## API Reference

### NumberField

The root component that manages the numeric value.

| Prop            | Type                       | Default | Description                                          |
| --------------- | -------------------------- | ------- | ---------------------------------------------------- |
| `modelValue`    | `number`                   | —       | The controlled value (v-model).                      |
| `defaultValue`  | `number`                   | —       | The default value for uncontrolled usage.            |
| `min`           | `number`                   | —       | Minimum allowed value.                               |
| `max`           | `number`                   | —       | Maximum allowed value.                               |
| `step`          | `number`                   | `1`     | Step increment.                                      |
| `disabled`      | `boolean`                  | `false` | Disable the number field.                            |
| `formatOptions` | `Intl.NumberFormatOptions` | —       | Number formatting options (e.g., currency, percent). |
| `locale`        | `string`                   | —       | Locale for number formatting.                        |
| `class`         | `string`                   | —       | Custom CSS classes.                                  |

#### Events

| Event               | Payload  | Description                     |
| ------------------- | -------- | ------------------------------- |
| `update:modelValue` | `number` | Emitted when the value changes. |

### NumberFieldContent

Layout wrapper for the input and buttons.

### NumberFieldInput

The numeric input element.

### NumberFieldIncrement

Button to increase the value by `step`.

### NumberFieldDecrement

Button to decrease the value by `step`.
