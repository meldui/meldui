## API Reference

### Props

| Prop                    | Type                                                                          | Default        | Description                                                     |
| ----------------------- | ----------------------------------------------------------------------------- | -------------- | --------------------------------------------------------------- |
| `modelValue`            | `number[]`                                                                    | —              | The controlled value (v-model). Array allows for range sliders. |
| `defaultValue`          | `number[]`                                                                    | —              | The default value for uncontrolled usage.                       |
| `variant`               | `'default' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'neutral'` | `'default'`    | Color variant.                                                  |
| `min`                   | `number`                                                                      | `0`            | Minimum value.                                                  |
| `max`                   | `number`                                                                      | `100`          | Maximum value.                                                  |
| `step`                  | `number`                                                                      | `1`            | Step increment.                                                 |
| `orientation`           | `'horizontal' \| 'vertical'`                                                  | `'horizontal'` | Slider orientation.                                             |
| `disabled`              | `boolean`                                                                     | `false`        | Disable the slider.                                             |
| `inverted`              | `boolean`                                                                     | `false`        | Invert the slider direction.                                    |
| `minStepsBetweenThumbs` | `number`                                                                      | —              | Minimum steps between thumbs for range sliders.                 |
| `class`                 | `string`                                                                      | —              | Custom CSS classes.                                             |

### Events

| Event               | Payload    | Description                              |
| ------------------- | ---------- | ---------------------------------------- |
| `update:modelValue` | `number[]` | Emitted when the value changes.          |
| `valueCommit`       | `number[]` | Emitted when the user finishes dragging. |
