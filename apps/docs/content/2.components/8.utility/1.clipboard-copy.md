## API Reference

### Props

| Prop      | Type     | Default | Description                                                                    |
| --------- | -------- | ------- | ------------------------------------------------------------------------------ |
| `value`   | `string` | â€”       | The text value to copy to the clipboard.                                       |
| `timeout` | `number` | `2000`  | Duration in milliseconds that the `copied` state remains `true` after copying. |

### Slots

| Slot      | Props                                   | Description                                                                              |
| --------- | --------------------------------------- | ---------------------------------------------------------------------------------------- |
| `default` | `{ copied: boolean, copy: () => void }` | Render function that receives the current copy state and a function to trigger the copy. |

### Exposed State

The component exposes the copy state through its default slot:

- **`copied`** -- `boolean` indicating whether the value was recently copied.
- **`copy`** -- `() => void` function that copies the `value` to the clipboard and sets `copied` to `true` for the configured `timeout` duration.
