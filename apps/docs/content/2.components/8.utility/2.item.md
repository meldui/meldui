## API Reference

### Item

The root container for a list item.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description                                            |
| --------- | ----- | ------------------------------------------------------ |
| `default` | —     | Item sub-components (`ItemIcon`, `ItemContent`, etc.). |

### ItemContent

Container for the label and description text.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### ItemLabel

The primary text label of the item.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### ItemDescription

Secondary descriptive text below the label.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### ItemIcon

Container for an icon displayed alongside the item content.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description                        |
| --------- | ----- | ---------------------------------- |
| `default` | —     | The icon component or SVG element. |
