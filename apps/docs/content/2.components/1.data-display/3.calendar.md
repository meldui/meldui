## API Reference

### Props

| Prop             | Type                                              | Default | Description                                   |
| ---------------- | ------------------------------------------------- | ------- | --------------------------------------------- |
| `modelValue`     | `DateValue`                                       | —       | The selected date (v-model).                  |
| `placeholder`    | `DateValue`                                       | Today   | The displayed month when no date is selected. |
| `layout`         | `'month-and-year' \| 'month-only' \| 'year-only'` | —       | Header selector layout.                       |
| `locale`         | `string`                                          | `'en'`  | BCP 47 locale tag for internationalization.   |
| `disabled`       | `boolean`                                         | `false` | Disable the calendar.                         |
| `readonly`       | `boolean`                                         | `false` | Make the calendar read-only.                  |
| `minValue`       | `DateValue`                                       | —       | Minimum selectable date.                      |
| `maxValue`       | `DateValue`                                       | —       | Maximum selectable date.                      |
| `numberOfMonths` | `number`                                          | `1`     | Number of months to display.                  |

### Events

| Event                | Payload     | Description                               |
| -------------------- | ----------- | ----------------------------------------- |
| `update:modelValue`  | `DateValue` | Emitted when the selected date changes.   |
| `update:placeholder` | `DateValue` | Emitted when the displayed month changes. |

### Sub-Components

The Calendar is composed of several sub-components that can be used individually for custom layouts:

- `CalendarHeader` — Header section with navigation
- `CalendarHeading` — Month/year heading with selectors
- `CalendarPrevButton` / `CalendarNextButton` — Navigation buttons
- `CalendarGrid` — Month grid container
- `CalendarGridHead` / `CalendarGridBody` — Grid sections
- `CalendarGridRow` — Row in the grid
- `CalendarHeadCell` — Day name header cells
- `CalendarCell` — Individual date cell container
- `CalendarCellTrigger` — Clickable date button
