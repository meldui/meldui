## API Reference

### Table

Root wrapper that provides scrolling and base table styles.

### TableHeader

The `<thead>` section.

### TableBody

The `<tbody>` section.

### TableFooter

The `<tfoot>` section.

### TableRow

A `<tr>` element with hover and selection styling.

### TableHead

A `<th>` header cell.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### TableCell

A `<td>` body cell.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### TableCaption

A `<caption>` element for table descriptions.

### TableEmpty

An empty state row spanning all columns.

| Prop      | Type     | Default | Description                |
| --------- | -------- | ------- | -------------------------- |
| `colspan` | `number` | `1`     | Number of columns to span. |

### Slots

All sub-components expose a `default` slot.
