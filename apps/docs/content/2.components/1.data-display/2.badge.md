## API Reference

### Props

| Prop      | Type                                                                                         | Default     | Description                          |
| --------- | -------------------------------------------------------------------------------------------- | ----------- | ------------------------------------ |
| `variant` | `'default' \| 'secondary' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'neutral'` | `'default'` | Color variant.                       |
| `shape`   | `'pill' \| 'tag'`                                                                            | `'pill'`    | Border radius style.                 |
| `outline` | `boolean`                                                                                    | `false`     | Use outline style instead of filled. |
| `as`      | `string \| Component`                                                                        | `'div'`     | The element to render as.            |

### Slots

| Slot      | Description                  |
| --------- | ---------------------------- |
| `default` | Badge content (text, icons). |
