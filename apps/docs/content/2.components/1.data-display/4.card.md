## API Reference

### Card

The root container.

| Prop    | Type     | Default | Description         |

### CardHeader

Header section with grid layout for title, description, and optional action.

### CardTitle

The card title, rendered as `<h3>`.

### CardDescription

The card description text.

### CardAction

Action area positioned in the top-right of the header.

### CardContent

Main content area.

### CardFooter

Footer section with flex layout.

### Slots

All sub-components expose a `default` slot for their content.
