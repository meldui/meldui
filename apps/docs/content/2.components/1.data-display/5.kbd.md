## API Reference

### Kbd

Displays a single keyboard key.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Description        |
| --------- | ------------------ |
| `default` | Key label or icon. |

### KbdGroup

Groups multiple `Kbd` elements with spacing.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Description       |
| --------- | ----------------- |
| `default` | `Kbd` components. |
