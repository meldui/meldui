## API Reference

### Props

| Prop             | Type                       | Default  | Description                                                           |
| ---------------- | -------------------------- | -------- | --------------------------------------------------------------------- |
| `date`           | `Date \| string \| number` | Required | The date to format. Accepts Date objects, ISO strings, or timestamps. |
| `autoUpdate`     | `boolean`                  | `false`  | Automatically refresh the relative time.                              |
| `updateInterval` | `number`                   | `30000`  | Update interval in milliseconds when `autoUpdate` is enabled.         |
