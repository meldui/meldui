## API Reference

### Timeline

The root container that manages item state.

| Prop              | Type                         | Default      | Description                                      |
| ----------------- | ---------------------------- | ------------ | ------------------------------------------------ |
| `orientation`     | `'vertical' \| 'horizontal'` | `'vertical'` | Timeline direction.                              |
| `variant`         | `'default' \| 'alternate'`   | `'default'`  | Layout variant.                                  |
| `activeIndex`     | `number`                     | `-1`         | Index of the active item (0-based, -1 for none). |
| `contentPosition` | `'start' \| 'end'`           | `'end'`      | Content position relative to the separator.      |

### TimelineItem

An individual entry in the timeline. Automatically receives status based on `activeIndex`.

### TimelineSeparator

Container for the dot and connector line.

### TimelineDot

Status indicator dot that changes color based on status (completed, active, pending).

| Prop   | Type                   | Default | Description |
| ------ | ---------------------- | ------- | ----------- |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'`  | Dot size.   |

### TimelineConnector

The line connecting dots between timeline items.

| Prop        | Type                             | Default     | Description     |
| ----------- | -------------------------------- | ----------- | --------------- |
| `thickness` | `'thin' \| 'default' \| 'thick'` | `'default'` | Line thickness. |

### TimelineContent

Content wrapper for each timeline item.

### TimelineHeader

Header section containing the title and time.

### TimelineTitle

Title text for the timeline item.

### TimelineDescription

Description text for the timeline item.
