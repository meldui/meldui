## API Reference

### Avatar

The root container for an avatar.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### AvatarImage

The image element inside the avatar.

| Prop  | Type     | Default | Description             |
| ----- | -------- | ------- | ----------------------- |
| `src` | `string` | —       | Image source URL.       |
| `alt` | `string` | —       | Alt text for the image. |

### AvatarFallback

Fallback content displayed when the image is unavailable.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Description                            |
| --------- | -------------------------------------- |
| `default` | Fallback content (typically initials). |

### AvatarGroup

Groups multiple avatars with overlapping layout and overflow.

| Prop          | Type                         | Default        | Description                                        |
| ------------- | ---------------------------- | -------------- | -------------------------------------------------- |
| `max`         | `number`                     | —              | Maximum avatars to show before overflow indicator. |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction.                                  |
| `spacing`     | `'sm' \| 'md' \| 'lg'`       | `'md'`         | Overlap spacing between avatars.                   |
| `reverse`     | `boolean`                    | `false`        | Reverse the stacking order.                        |

#### Slots

| Slot       | Props               | Description                |
| ---------- | ------------------- | -------------------------- |
| `default`  | —                   | Avatar components.         |
| `overflow` | `{ count: number }` | Custom overflow indicator. |
