## API Reference

### Drawer

The root container that manages the drawer open/closed state.

| Prop                    | Type      | Default | Description                                                |
| ----------------------- | --------- | ------- | ---------------------------------------------------------- |
| `open`                  | `boolean` | `false` | The controlled open state. Use with `v-model`.             |
| `shouldScaleBackground` | `boolean` | `true`  | Whether the background should scale when the drawer opens. |

#### Events

| Event         | Payload   | Description                          |
| ------------- | --------- | ------------------------------------ |
| `update:open` | `boolean` | Emitted when the open state changes. |

### DrawerTrigger

The element that opens the drawer.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### DrawerContent

The sliding panel content container.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### DrawerHeader

Container for the drawer title and description.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### DrawerTitle

The accessible title of the drawer.

### DrawerDescription

The accessible description that provides additional context.

### DrawerFooter

Container for the drawer action buttons.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### DrawerClose

A button that closes the drawer.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |
