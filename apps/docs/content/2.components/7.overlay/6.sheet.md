## API Reference

### Sheet

The root container that manages the sheet open/closed state.

| Prop          | Type      | Default | Description                                    |
| ------------- | --------- | ------- | ---------------------------------------------- |
| `open`        | `boolean` | `false` | The controlled open state. Use with `v-model`. |
| `defaultOpen` | `boolean` | `false` | The default open state when uncontrolled.      |

#### Events

| Event         | Payload   | Description                          |
| ------------- | --------- | ------------------------------------ |
| `update:open` | `boolean` | Emitted when the open state changes. |

### SheetTrigger

The element that opens the sheet.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### SheetContent

The sliding panel content container.

| Prop    | Type                                     | Default   | Description                                      |
| ------- | ---------------------------------------- | --------- | ------------------------------------------------ |
| `side`  | `'top' \| 'right' \| 'bottom' \| 'left'` | `'right'` | The edge of the screen the sheet slides in from. |
| `class` | `string`                                 | —         | Custom CSS classes.                              |

### SheetHeader

Container for the sheet title and description.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### SheetTitle

The accessible title of the sheet.

### SheetDescription

The accessible description that provides additional context.

### SheetFooter

Container for the sheet action buttons.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### SheetClose

A button that closes the sheet.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |
