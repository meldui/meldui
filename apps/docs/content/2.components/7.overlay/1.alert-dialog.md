## API Reference

### AlertDialog

The root container that manages the dialog open/closed state.

### AlertDialogTrigger

The element that opens the alert dialog.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### AlertDialogContent

The modal overlay and content container.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### AlertDialogHeader

Container for the dialog title and description.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### AlertDialogTitle

The accessible title of the alert dialog.

### AlertDialogDescription

The accessible description that provides additional context.

### AlertDialogFooter

Container for the dialog action buttons.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### AlertDialogAction

The primary action button (e.g., "Confirm", "Delete").

### AlertDialogCancel

The secondary action button that closes the dialog without performing the action.
