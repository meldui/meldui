## API Reference

### Dialog

The root container that manages the dialog open/closed state.

| Prop          | Type      | Default | Description                                    |
| ------------- | --------- | ------- | ---------------------------------------------- |
| `open`        | `boolean` | `false` | The controlled open state. Use with `v-model`. |
| `defaultOpen` | `boolean` | `false` | The default open state when uncontrolled.      |

#### Events

| Event         | Payload   | Description                          |
| ------------- | --------- | ------------------------------------ |
| `update:open` | `boolean` | Emitted when the open state changes. |

### DialogTrigger

The element that opens the dialog.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### DialogContent

The modal overlay and content container.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### DialogHeader

Container for the dialog title and description.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### DialogTitle

The accessible title of the dialog.

### DialogDescription

The accessible description that provides additional context.

### DialogFooter

Container for the dialog action buttons.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### DialogClose

A button that closes the dialog.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |
