## API Reference

### HoverCard

The root container that manages the hover card open/closed state.

| Prop         | Type     | Default | Description                                                     |
| ------------ | -------- | ------- | --------------------------------------------------------------- |
| `openDelay`  | `number` | `700`   | Delay in milliseconds before the card opens on hover.           |
| `closeDelay` | `number` | `300`   | Delay in milliseconds before the card closes when hover leaves. |

### HoverCardTrigger

The element that triggers the hover card on mouse enter.

| Prop      | Type      | Default | Description                                  |
| --------- | --------- | ------- | -------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a link. |

### HoverCardContent

The floating content panel displayed on hover.

| Prop         | Type                           | Default    | Description                                |
| ------------ | ------------------------------ | ---------- | ------------------------------------------ |
| `class`      | `string`                       | —          | Custom CSS classes.                        |
| `align`      | `'start' \| 'center' \| 'end'` | `'center'` | Alignment relative to the trigger.         |
| `sideOffset` | `number`                       | `4`        | Offset from the trigger element in pixels. |

#### Slots

| Slot      | Props | Description             |
| --------- | ----- | ----------------------- |
| `default` | —     | The hover card content. |
