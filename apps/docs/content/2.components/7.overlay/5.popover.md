## API Reference

### Popover

The root container that manages the popover open/closed state.

| Prop          | Type      | Default | Description                                    |
| ------------- | --------- | ------- | ---------------------------------------------- |
| `open`        | `boolean` | `false` | The controlled open state. Use with `v-model`. |
| `defaultOpen` | `boolean` | `false` | The default open state when uncontrolled.      |

#### Events

| Event         | Payload   | Description                          |
| ------------- | --------- | ------------------------------------ |
| `update:open` | `boolean` | Emitted when the open state changes. |

### PopoverTrigger

The element that toggles the popover.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### PopoverContent

The floating content panel.

| Prop         | Type                                     | Default    | Description                                     |
| ------------ | ---------------------------------------- | ---------- | ----------------------------------------------- |
| `class`      | `string`                                 | —          | Custom CSS classes.                             |
| `align`      | `'start' \| 'center' \| 'end'`           | `'center'` | Alignment relative to the trigger.              |
| `sideOffset` | `number`                                 | `4`        | Offset from the trigger element in pixels.      |
| `side`       | `'top' \| 'right' \| 'bottom' \| 'left'` | `'bottom'` | The preferred side of the trigger to render on. |

#### Slots

| Slot      | Props | Description          |
| --------- | ----- | -------------------- |
| `default` | —     | The popover content. |
