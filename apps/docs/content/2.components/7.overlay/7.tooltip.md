## API Reference

### TooltipProvider

Wraps your application or a section of it to provide shared tooltip configuration.

| Prop                | Type     | Default | Description                                                           |
| ------------------- | -------- | ------- | --------------------------------------------------------------------- |
| `delayDuration`     | `number` | `700`   | Delay in milliseconds before tooltips open.                           |
| `skipDelayDuration` | `number` | `300`   | Time in milliseconds before the delay resets after leaving a trigger. |

### Tooltip

The root container for a single tooltip instance.

| Prop            | Type      | Default | Description                                     |
| --------------- | --------- | ------- | ----------------------------------------------- |
| `open`          | `boolean` | —       | The controlled open state. Use with `v-model`.  |
| `defaultOpen`   | `boolean` | `false` | The default open state when uncontrolled.       |
| `delayDuration` | `number`  | —       | Override the provider's delay for this tooltip. |

#### Events

| Event         | Payload   | Description                          |
| ------------- | --------- | ------------------------------------ |
| `update:open` | `boolean` | Emitted when the open state changes. |

### TooltipTrigger

The element that triggers the tooltip on hover or focus.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### TooltipContent

The floating tooltip content.

| Prop         | Type                                     | Default    | Description                                     |
| ------------ | ---------------------------------------- | ---------- | ----------------------------------------------- |
| `class`      | `string`                                 | —          | Custom CSS classes.                             |
| `side`       | `'top' \| 'right' \| 'bottom' \| 'left'` | `'top'`    | The preferred side of the trigger to render on. |
| `sideOffset` | `number`                                 | `4`        | Offset from the trigger element in pixels.      |
| `align`      | `'start' \| 'center' \| 'end'`           | `'center'` | Alignment relative to the trigger.              |

#### Slots

| Slot      | Props | Description                                          |
| --------- | ----- | ---------------------------------------------------- |
| `default` | —     | The tooltip content (text, formatted content, etc.). |
