## API Reference

### Props

| Prop         | Type                                                                          | Default     | Description                        |
| ------------ | ----------------------------------------------------------------------------- | ----------- | ---------------------------------- |
| `modelValue` | `number`                                                                      | `0`         | Current progress value.            |
| `max`        | `number`                                                                      | `100`       | Maximum progress value.            |
| `variant`    | `'default' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'neutral'` | `'default'` | Color variant of the progress bar. |
