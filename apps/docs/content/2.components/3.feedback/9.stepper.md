## API Reference

### Stepper

The root component that manages step state. Bind the active step with `v-model`.

| Prop          | Type                         | Default        | Description                                  |
| ------------- | ---------------------------- | -------------- | -------------------------------------------- |
| `modelValue`  | `number`                     | —              | The currently active step (1-based).         |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction of the stepper.             |
| `linear`      | `boolean`                    | `false`        | When true, steps must be completed in order. |

| Event               | Payload  | Description                           |
| ------------------- | -------- | ------------------------------------- |
| `update:modelValue` | `number` | Emitted when the active step changes. |

### StepperItem

An individual step in the stepper. Automatically receives state (`active`, `completed`, `inactive`) based on the current model value.

| Prop       | Type      | Default | Description                         |
| ---------- | --------- | ------- | ----------------------------------- |
| `step`     | `number`  | —       | The step number (1-based).          |
| `disabled` | `boolean` | `false` | Disable interaction with this step. |

### StepperTrigger

Clickable area that allows users to navigate to a step.

### StepperIndicator

Visual indicator showing the step number or a custom icon. Styled based on the step state.

| Slot      | Description                       |
| --------- | --------------------------------- |
| `default` | Step number, icon, or check mark. |

### StepperSeparator

The connecting line between steps. Color changes based on completion state.

### StepperTitle

Title label for a step.

| Slot      | Description          |
| --------- | -------------------- |
| `default` | The step title text. |

### StepperDescription

Description text below the step title.

| Slot      | Description                |
| --------- | -------------------------- |
| `default` | The step description text. |
