## API Reference

### Empty

The root container with centered layout and dashed border styling.

| Slot      | Description                                                   |
| --------- | ------------------------------------------------------------- |
| `default` | Content including `EmptyHeader` and optional action elements. |

### EmptyHeader

Groups the media, title, and description together.

| Slot      | Description                                                            |
| --------- | ---------------------------------------------------------------------- |
| `default` | Typically contains `EmptyMedia`, `EmptyTitle`, and `EmptyDescription`. |

### EmptyMedia

Container for the empty state icon or illustration.

| Prop      | Type                  | Default     | Description                                            |
| --------- | --------------------- | ----------- | ------------------------------------------------------ |
| `variant` | `'default' \| 'icon'` | `'default'` | Use `'icon'` for a rounded background behind the icon. |

| Slot      | Description               |
| --------- | ------------------------- |
| `default` | An icon or image element. |

### EmptyTitle

Heading text for the empty state.

| Slot      | Description     |
| --------- | --------------- |
| `default` | The title text. |

### EmptyDescription

Body text explaining the empty state or suggesting next steps.

| Slot      | Description           |
| --------- | --------------------- |
| `default` | The description text. |

### EmptyContent

Optional wrapper for additional content below the header (e.g. action buttons).

| Slot      | Description                                  |
| --------- | -------------------------------------------- |
| `default` | Additional content such as buttons or links. |
