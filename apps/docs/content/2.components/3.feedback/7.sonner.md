## Usage

Add the `Toaster` component to your app root, then use the `toast` function to trigger notifications.

## API Reference

### Toaster

The toast container component. Place it once at the root of your application.

All props from `vue-sonner`'s `ToasterProps` are supported, including:

| Prop          | Type                                                                                              | Default          | Description                                  |
| ------------- | ------------------------------------------------------------------------------------------------- | ---------------- | -------------------------------------------- |
| `position`    | `'top-left' \| 'top-right' \| 'bottom-left' \| 'bottom-right' \| 'top-center' \| 'bottom-center'` | `'bottom-right'` | Position of the toast stack.                 |
| `expand`      | `boolean`                                                                                         | `false`          | Expand toasts by default.                    |
| `duration`    | `number`                                                                                          | `4000`           | Auto-dismiss duration in milliseconds.       |
| `closeButton` | `boolean`                                                                                         | `false`          | Show a close button on each toast.           |
| `richColors`  | `boolean`                                                                                         | `false`          | Use richer colors for different toast types. |

### toast()

Function to create toast notifications. Import it from `@meldui/vue`.

| Method                            | Description                                               |
| --------------------------------- | --------------------------------------------------------- |
| `toast(message)`                  | Show a default toast.                                     |
| `toast.success(message)`          | Show a success toast.                                     |
| `toast.error(message)`            | Show an error toast.                                      |
| `toast.warning(message)`          | Show a warning toast.                                     |
| `toast.info(message)`             | Show an info toast.                                       |
| `toast.loading(message)`          | Show a loading toast with a spinner.                      |
| `toast.promise(promise, options)` | Show loading, success, or error states tied to a promise. |

### useVueSonner

Composable that provides reactive access to the list of active toasts.
