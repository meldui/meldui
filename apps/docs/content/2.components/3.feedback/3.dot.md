## API Reference

### Props

| Prop      | Type                                                                          | Default     | Description                          |
| --------- | ----------------------------------------------------------------------------- | ----------- | ------------------------------------ |
| `variant` | `'primary' \| 'success' \| 'warning' \| 'destructive' \| 'info' \| 'neutral'` | `'primary'` | Color variant of the dot.            |
| `size`    | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`                                        | `'md'`      | Size of the dot.                     |
| `effect`  | `'pulse' \| 'ping' \| undefined`                                              | `undefined` | Animation effect applied to the dot. |
