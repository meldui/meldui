## API Reference

### CircularProgress

The root component that provides progress context to child components.

| Prop            | Type                                                                          | Default      | Description                                                    |
| --------------- | ----------------------------------------------------------------------------- | ------------ | -------------------------------------------------------------- |
| `value`         | `number \| null`                                                              | `0`          | Current progress value. Set to `null` for indeterminate state. |
| `min`           | `number`                                                                      | `0`          | Minimum value.                                                 |
| `max`           | `number`                                                                      | `100`        | Maximum value.                                                 |
| `size`          | `number`                                                                      | `48`         | Diameter of the circle in pixels.                              |
| `thickness`     | `number`                                                                      | `4`          | Stroke width of the circle in pixels.                          |
| `variant`       | `'default' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'neutral'` | `'default'`  | Color variant.                                                 |
| `getValueLabel` | `(value: number, max: number) => string`                                      | `percentage` | Custom function to format the value label.                     |

### CircularProgressIndicator

SVG container for the track and range arcs.

### CircularProgressTrack

The background circle track.

### CircularProgressRange

The filled arc representing the current progress.

### CircularProgressLabel

Displays the formatted value label (e.g. "75%") centered inside the circle.
