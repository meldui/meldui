## API Reference

### Alert

The root container for the alert message.

| Prop      | Type                                                                          | Default     | Description                     |
| --------- | ----------------------------------------------------------------------------- | ----------- | ------------------------------- |
| `variant` | `'default' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'neutral'` | `'default'` | The color variant of the alert. |

#### Slots

| Slot      | Description                                                                     |
| --------- | ------------------------------------------------------------------------------- |
| `default` | Alert content including an optional icon, `AlertTitle`, and `AlertDescription`. |

### AlertTitle

Renders the alert heading text.

| Slot      | Description     |
| --------- | --------------- |
| `default` | The title text. |

### AlertDescription

Renders the alert body text.

| Slot      | Description           |
| --------- | --------------------- |
| `default` | The description text. |
