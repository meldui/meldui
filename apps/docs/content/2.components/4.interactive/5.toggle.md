## API Reference

### Props

| Prop             | Type                        | Default     | Description                                       |
| ---------------- | --------------------------- | ----------- | ------------------------------------------------- |
| `variant`        | `'default' \| 'outline'`    | `'default'` | The visual style of the toggle.                   |
| `size`           | `'default' \| 'sm' \| 'lg'` | `'default'` | The size of the toggle.                           |
| `pressed`        | `boolean`                   | `false`     | The controlled pressed state. Use with `v-model`. |
| `defaultPressed` | `boolean`                   | `false`     | The default pressed state when uncontrolled.      |
| `disabled`       | `boolean`                   | `false`     | Whether the toggle is disabled.                   |

Based on [Reka UI Toggle](https://reka-ui.com/docs/components/toggle).

### Events

| Event            | Payload   | Description                             |
| ---------------- | --------- | --------------------------------------- |
| `update:pressed` | `boolean` | Emitted when the pressed state changes. |
