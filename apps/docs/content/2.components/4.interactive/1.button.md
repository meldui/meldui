## API Reference

### Props

| Prop       | Type                                                                                                              | Default     | Description                                    |
| ---------- | ----------------------------------------------------------------------------------------------------------------- | ----------- | ---------------------------------------------- |
| `variant`  | `'default' \| 'secondary' \| 'destructive' \| 'success' \| 'warning' \| 'info' \| 'outline' \| 'ghost' \| 'link'` | `'default'` | The visual style of the button.                |
| `size`     | `'default' \| 'sm' \| 'lg' \| 'icon' \| 'icon-sm' \| 'icon-lg'`                                                   | `'default'` | The size of the button.                        |
| `as`       | `string \| Component`                                                                                             | `'button'`  | The element or component to render as.         |
| `asChild`  | `boolean`                                                                                                         | `false`     | Render as a child element instead of a button. |
| `disabled` | `boolean`                                                                                                         | `false`     | Disable the button.                            |

### Events

| Event   | Payload      | Description                         |
| ------- | ------------ | ----------------------------------- |
| `click` | `MouseEvent` | Emitted when the button is clicked. |
