## API Reference

### Carousel

The root carousel container. Built on top of [Embla Carousel](https://www.embla-carousel.com/).

| Prop          | Type                         | Default        | Description                           |
| ------------- | ---------------------------- | -------------- | ------------------------------------- |
| `opts`        | `EmblaOptionsType`           | —              | Embla Carousel options object.        |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | The scroll direction of the carousel. |
| `plugins`     | `EmblaPluginType[]`          | —              | Embla Carousel plugins array.         |

### CarouselContent

Container for carousel items. Handles the sliding viewport.

### CarouselItem

Individual slide within the carousel.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### CarouselPrevious

Button to navigate to the previous slide.

### CarouselNext

Button to navigate to the next slide.
