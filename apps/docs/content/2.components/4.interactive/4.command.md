## API Reference

### Command

The root command container.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

### CommandInput

Search input field for filtering command items.

| Prop          | Type     | Default | Description                     |
| ------------- | -------- | ------- | ------------------------------- |
| `placeholder` | `string` | —       | Placeholder text for the input. |

### CommandList

Scrollable list container for command groups and items.

### CommandGroup

Groups related command items under an optional heading.

| Prop      | Type     | Default | Description                      |
| --------- | -------- | ------- | -------------------------------- |
| `heading` | `string` | —       | Label displayed above the group. |

### CommandItem

An individual selectable command item.

| Prop       | Type      | Default | Description                                 |
| ---------- | --------- | ------- | ------------------------------------------- |
| `value`    | `string`  | —       | The value used for filtering and selection. |
| `disabled` | `boolean` | `false` | Whether the item is disabled.               |

### CommandEmpty

Displayed when no results match the current search query.

### CommandSeparator

A visual divider between command groups.

### CommandDialog

A dialog wrapper around the command palette.

| Prop          | Type     | Default | Description                    |
| ------------- | -------- | ------- | ------------------------------ |
| `title`       | `string` | —       | Accessible dialog title.       |
| `description` | `string` | —       | Accessible dialog description. |
