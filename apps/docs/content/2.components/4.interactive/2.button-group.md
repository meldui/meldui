## API Reference

### ButtonGroup

The root container that groups multiple `Button` components together.

| Prop          | Type                         | Default        | Description                                  |
| ------------- | ---------------------------- | -------------- | -------------------------------------------- |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | The layout direction of the button group.    |
| `variant`     | `string`                     | —              | Apply a shared variant to all child buttons. |
| `size`        | `string`                     | —              | Apply a shared size to all child buttons.    |
| `class`       | `string`                     | —              | Custom CSS classes.                          |

### Slots

| Slot      | Props | Description                            |
| --------- | ----- | -------------------------------------- |
| `default` | —     | `Button` components to group together. |
