## API Reference

### Accordion

The root container for accordion items.

| Prop           | Type                     | Default    | Description                                                 |
| -------------- | ------------------------ | ---------- | ----------------------------------------------------------- |
| `type`         | `'single' \| 'multiple'` | `'single'` | Whether one or multiple items can be open at the same time. |
| `collapsible`  | `boolean`                | `false`    | When `type` is `'single'`, allows closing the open item.    |
| `defaultValue` | `string \| string[]`     | —          | The default open item(s).                                   |
| `modelValue`   | `string \| string[]`     | —          | The controlled open item(s). Use with `v-model`.            |
| `disabled`     | `boolean`                | `false`    | Whether the entire accordion is disabled.                   |

#### Events

| Event               | Payload              | Description                           |
| ------------------- | -------------------- | ------------------------------------- |
| `update:modelValue` | `string \| string[]` | Emitted when the open item(s) change. |

### AccordionItem

A single collapsible section within the accordion.

| Prop       | Type      | Default | Description                           |
| ---------- | --------- | ------- | ------------------------------------- |
| `value`    | `string`  | —       | A unique value identifying this item. |
| `disabled` | `boolean` | `false` | Whether this item is disabled.        |

### AccordionTrigger

The button that toggles the visibility of the associated content.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description                |
| --------- | ----- | -------------------------- |
| `default` | —     | The trigger label content. |

### AccordionContent

The collapsible content area revealed when the trigger is activated.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description                           |
| --------- | ----- | ------------------------------------- |
| `default` | —     | The content to display when expanded. |
