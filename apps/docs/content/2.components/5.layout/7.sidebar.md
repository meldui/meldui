## Overview

The Sidebar component is a comprehensive layout primitive for building application navigation. It provides a full set of sub-components for constructing sidebars with headers, footers, menus, groups, and collapsible behavior.

Due to its complexity, no live preview is provided. See the sub-components reference below for building your own sidebar.

## Sub-components

The Sidebar system is composed of the following parts:

- **SidebarProvider** -- Wraps your application layout and provides sidebar state context.
- **Sidebar** -- The main sidebar container.
- **SidebarHeader** -- Top section of the sidebar (logo, branding).
- **SidebarContent** -- Main scrollable content area.
- **SidebarFooter** -- Bottom section of the sidebar (user info, actions).
- **SidebarGroup** -- Groups related menu items together.
- **SidebarGroupLabel** -- Label for a sidebar group.
- **SidebarGroupContent** -- Content container within a group.
- **SidebarMenu** -- Container for menu items within a group.
- **SidebarMenuItem** -- Individual menu item wrapper.
- **SidebarMenuButton** -- Clickable button within a menu item.
- **SidebarMenuAction** -- Secondary action button within a menu item.
- **SidebarMenuBadge** -- Badge indicator on a menu item.
- **SidebarMenuSub** -- Container for nested sub-menu items.
- **SidebarMenuSubItem** -- Individual sub-menu item wrapper.
- **SidebarMenuSubButton** -- Clickable button within a sub-menu item.
- **SidebarSeparator** -- Visual divider within the sidebar.
- **SidebarInset** -- Main content area that adjusts when the sidebar is open.
- **SidebarInput** -- Search input within the sidebar.
- **SidebarRail** -- Narrow rail visible when the sidebar is collapsed.
- **SidebarTrigger** -- Button that toggles the sidebar open/closed.
- **SidebarMenuSkeleton** -- Loading placeholder for menu items.

## API Reference

### Sidebar

| Prop          | Type                                 | Default       | Description                                   |
| ------------- | ------------------------------------ | ------------- | --------------------------------------------- |
| `side`        | `'left' \| 'right'`                  | `'left'`      | Which side the sidebar appears on.            |
| `variant`     | `'sidebar' \| 'floating' \| 'inset'` | `'sidebar'`   | The visual variant of the sidebar.            |
| `collapsible` | `'offcanvas' \| 'icon' \| 'none'`    | `'offcanvas'` | How the sidebar collapses on smaller screens. |

### SidebarMenuButton

| Prop       | Type                        | Default     | Description                                       |
| ---------- | --------------------------- | ----------- | ------------------------------------------------- |
| `variant`  | `'default' \| 'outline'`    | `'default'` | The visual style of the button.                   |
| `size`     | `'default' \| 'sm' \| 'lg'` | `'default'` | The size of the button.                           |
| `isActive` | `boolean`                   | `false`     | Whether the button is in an active state.         |
| `tooltip`  | `string`                    | â€”           | Tooltip text shown when the sidebar is collapsed. |

### useSidebar Composable

Access sidebar state and controls from any descendant component.

```vue
<script setup>
import { useSidebar } from '@meldui/vue'

const { open, toggleSidebar } = useSidebar()
</script>
```

| Property        | Type           | Description                            |
| --------------- | -------------- | -------------------------------------- |
| `open`          | `Ref<boolean>` | Whether the sidebar is currently open. |
| `toggleSidebar` | `() => void`   | Toggle the sidebar open/closed.        |
