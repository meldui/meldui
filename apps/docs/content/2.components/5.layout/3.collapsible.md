## API Reference

### Collapsible

The root container that manages the open/closed state.

| Prop          | Type      | Default | Description                                    |
| ------------- | --------- | ------- | ---------------------------------------------- |
| `open`        | `boolean` | `false` | The controlled open state. Use with `v-model`. |
| `defaultOpen` | `boolean` | `false` | The default open state when uncontrolled.      |
| `disabled`    | `boolean` | `false` | Whether the collapsible is disabled.           |

#### Events

| Event         | Payload   | Description                          |
| ------------- | --------- | ------------------------------------ |
| `update:open` | `boolean` | Emitted when the open state changes. |

### CollapsibleTrigger

The element that toggles the visibility of the collapsible content.

| Prop      | Type      | Default | Description                                    |
| --------- | --------- | ------- | ---------------------------------------------- |
| `asChild` | `boolean` | `false` | Render as a child element instead of a button. |

### CollapsibleContent

The content area that is shown or hidden.

| Prop    | Type     | Default | Description         |
| ------- | -------- | ------- | ------------------- |
| `class` | `string` | —       | Custom CSS classes. |

#### Slots

| Slot      | Props | Description              |
| --------- | ----- | ------------------------ |
| `default` | —     | The collapsible content. |
