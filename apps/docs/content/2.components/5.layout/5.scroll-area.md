## API Reference

### ScrollArea

The main scrollable container with custom-styled scrollbars.

| Prop    | Type                                        | Default   | Description                             |
| ------- | ------------------------------------------- | --------- | --------------------------------------- |
| `class` | `string`                                    | —         | Custom CSS classes.                     |
| `type`  | `'auto' \| 'always' \| 'scroll' \| 'hover'` | `'hover'` | Controls when the scrollbar is visible. |

#### Slots

| Slot      | Props | Description             |
| --------- | ----- | ----------------------- |
| `default` | —     | The scrollable content. |

### ScrollBar

Custom scrollbar element displayed within the scroll area.

| Prop          | Type                         | Default      | Description              |
| ------------- | ---------------------------- | ------------ | ------------------------ |
| `orientation` | `'horizontal' \| 'vertical'` | `'vertical'` | The scrollbar direction. |
| `class`       | `string`                     | —            | Custom CSS classes.      |
