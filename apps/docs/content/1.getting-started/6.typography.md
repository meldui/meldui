MeldUI uses a clean, modern typography system based on system fonts with optional custom font support.

## Default Fonts

MeldUI ships with two font families configured:

| Font Family | Usage                          | Default Stack                |
| ----------- | ------------------------------ | ---------------------------- |
| `font-sans` | Body text, UI elements         | Inter, system-ui, sans-serif |
| `font-mono` | Code blocks, technical content | JetBrains Mono, ui-monospace |

## Configuring Fonts

Fonts are configured in your CSS using Tailwind CSS v4's `@theme` directive:

```css
@theme inline {
  --font-sans:
    'Inter', ui-sans-serif, system-ui, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji',
    'Segoe UI Symbol', 'Noto Color Emoji';
  --font-mono:
    'JetBrains Mono', ui-monospace, SFMono-Regular, 'SF Mono', Menlo, Consolas, 'Liberation Mono',
    monospace;
}
```

## Loading Custom Fonts

To use a specific font like Inter, add it to your HTML:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
  rel="stylesheet"
/>
```

Or install via npm:

```bash
pnpm add @fontsource/inter
```

```typescript
// main.ts
import '@fontsource/inter/400.css'
import '@fontsource/inter/500.css'
import '@fontsource/inter/600.css'
import '@fontsource/inter/700.css'
```

## Text Colors

MeldUI provides semantic text colors through CSS variables:

| Class                   | Usage                 |
| ----------------------- | --------------------- |
| `text-foreground`       | Primary text          |
| `text-muted-foreground` | Secondary/subtle text |
| `text-primary`          | Brand-colored text    |
| `text-destructive`      | Error/danger text     |
| `text-success`          | Success text          |
| `text-warning`          | Warning text          |
| `text-info`             | Informational text    |

## Font Weights

MeldUI components use these weight conventions:

| Weight | Class           | Usage                       |
| ------ | --------------- | --------------------------- |
| 400    | `font-normal`   | Body text                   |
| 500    | `font-medium`   | Labels, navigation, buttons |
| 600    | `font-semibold` | Headings, emphasis          |
| 700    | `font-bold`     | Page titles                 |
