## What is MeldUI?

MeldUI is an internal design system monorepo that provides a comprehensive set of Vue 3 components built on top of [shadcn-vue](https://www.shadcn-vue.com/). It includes:

- **70+ base UI components** — buttons, forms, dialogs, tables, navigation, and more
- **8 composite component families** — DataTable, DateRangePicker, MultiSelect, Timeline, and others
- **Chart components** — Area, Bar, Line, Pie, Donut, Radar, Scatter, Heatmap, and Mixed charts
- **Icon system** — 5000+ Tabler Icons wrapped with design system defaults via `@meldui/tabler-vue`
- **Tailwind CSS v4** — Modern CSS-first configuration with full dark mode support
- **Fully customizable** — All components are copied into your project for complete control

## Packages

| Package              | Description                             |
| -------------------- | --------------------------------------- |
| `@meldui/vue`        | Core component library (70+ components) |
| `@meldui/tabler-vue` | 5000+ Tabler icons with custom defaults |
| `@meldui/charts-vue` | Chart components built on ECharts       |

## Why MeldUI?

MeldUI provides a consistent, well-documented component library for internal engineering teams:

- **Consistency** — Shared design tokens, spacing, and interaction patterns
- **Quality** — Accessible, tested components with comprehensive API documentation
- **Productivity** — Copy-paste-ready patterns for common UI scenarios
- **Customization** — Full source access to every component

## Quick Start

```bash
pnpm add @meldui/vue @meldui/tabler-vue
```

```vue
<script setup lang="ts">
import { Button } from '@meldui/vue'
import { IconCheck } from '@meldui/tabler-vue'
</script>

<template>
  <Button>
    <IconCheck />
    Get Started
  </Button>
</template>
```

Follow the [Installation guide](/getting-started/installation) for the full setup.
