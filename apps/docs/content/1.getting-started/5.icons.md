MeldUI provides 5000+ icons through `@meldui/tabler-vue`, a wrapper around [Tabler Icons](https://tabler.io/icons) with custom defaults.

## Installation

```bash
pnpm add @meldui/tabler-vue
```

The package is automatically installed as a peer dependency of `@meldui/vue`.

## Usage

```vue
<script setup lang="ts">
import { IconUser, IconSettings, IconCheck } from '@meldui/tabler-vue'
</script>

<template>
  <!-- Default: 24px, stroke 1.5 -->
  <IconUser />

  <!-- Custom size -->
  <IconSettings :size="32" />

  <!-- Custom stroke -->
  <IconCheck :stroke="2" />
</template>
```

## Defaults

| Prop     | Default | Description                        |
| -------- | ------- | ---------------------------------- |
| `size`   | `24`    | Icon size in pixels                |
| `stroke` | `1.5`   | Stroke width (Tabler default is 2) |

MeldUI uses a stroke weight of 1.5 (lighter than Tabler's default of 2) for a cleaner appearance.

## Color

Icons use `currentColor` and inherit from the parent's text color:

```vue
<template>
  <!-- Icon inherits red from parent -->
  <span class="text-red-500">
    <IconX />
  </span>

  <!-- Color directly on the icon -->
  <IconCheck class="text-green-500" />

  <!-- Works with MeldUI theme colors -->
  <IconInfo class="text-primary" />
</template>
```

## With Buttons

Icons work seamlessly with MeldUI Button:

```vue
<script setup lang="ts">
import { Button } from '@meldui/vue'
import { IconMail, IconPlus } from '@meldui/tabler-vue'
</script>

<template>
  <!-- Icon + text -->
  <Button>
    <IconMail />
    Send Email
  </Button>

  <!-- Icon only -->
  <Button variant="outline" size="icon">
    <IconPlus />
  </Button>
</template>
```

## Naming Convention

All icons use the `Icon` prefix followed by the icon name in PascalCase:

| Icon Name     | Import             |
| ------------- | ------------------ |
| x             | `IconX`            |
| check         | `IconCheck`        |
| user          | `IconUser`         |
| settings      | `IconSettings`     |
| chevron-right | `IconChevronRight` |
| arrow-left    | `IconArrowLeft`    |

## Browsing Icons

Browse all 5000+ available icons at [tabler.io/icons](https://tabler.io/icons).

## Tree Shaking

Only the icons you import are included in your bundle. Each icon is an individual export, ensuring perfect tree-shaking:

```typescript
// Only IconUser and IconCheck are bundled
import { IconUser, IconCheck } from '@meldui/tabler-vue'
```
