## Sub-components

| Component          | Description                                                  |
| ------------------ | ------------------------------------------------------------ |
| `Command`          | Root container                                               |
| `CommandDialog`    | Command palette in a modal. Props: `title`, `description`    |
| `CommandInput`     | Search input with icon                                       |
| `CommandList`      | Scrollable list container                                    |
| `CommandGroup`     | Group of items. Props: `heading`                             |
| `CommandItem`      | Selectable item. Props: `value`, `disabled`. Event: `select` |
| `CommandEmpty`     | Empty state when no results match                            |
| `CommandSeparator` | Visual separator between groups                              |
| `CommandShortcut`  | Keyboard shortcut display                                    |
