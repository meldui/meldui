## Sub-components

| Component             | Description                                                                |
| --------------------- | -------------------------------------------------------------------------- |
| `ResizablePanelGroup` | Root container. Props: `direction` ('horizontal' \| 'vertical')            |
| `ResizablePanel`      | Resizable panel. Props: `defaultSize`, `minSize`, `maxSize`, `collapsible` |
| `ResizableHandle`     | Drag handle between panels. Props: `withHandle` (shows grip icon)          |
