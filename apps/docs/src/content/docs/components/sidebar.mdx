## Complex Example

A full sidebar with collapsible project groups, nested sub-menus (`SidebarMenuSub`), badge counts (`SidebarMenuBadge`), header, footer, and a toggle trigger. Uses `Collapsible` to expand/collapse groups with a chevron indicator.

## Sub-components

| Component              | Description                                                             |
| ---------------------- | ----------------------------------------------------------------------- |
| `SidebarProvider`      | Root state provider. Props: `defaultOpen`, `open`. Event: `update:open` |
| `Sidebar`              | Main sidebar container. Props: `side`, `variant`, `collapsible`         |
| `SidebarContent`       | Scrollable content area                                                 |
| `SidebarHeader`        | Top section                                                             |
| `SidebarFooter`        | Bottom section                                                          |
| `SidebarGroup`         | Section grouping                                                        |
| `SidebarGroupLabel`    | Group heading                                                           |
| `SidebarGroupContent`  | Group content                                                           |
| `SidebarGroupAction`   | Action button in group header                                           |
| `SidebarMenu`          | Menu container                                                          |
| `SidebarMenuItem`      | Menu item wrapper                                                       |
| `SidebarMenuButton`    | Clickable menu button. Props: `variant`, `size`, `tooltip`              |
| `SidebarMenuSub`       | Nested sub-menu                                                         |
| `SidebarMenuSubItem`   | Sub-menu item                                                           |
| `SidebarMenuSubButton` | Sub-menu button                                                         |
| `SidebarMenuAction`    | Action within a menu item                                               |
| `SidebarMenuBadge`     | Badge indicator                                                         |
| `SidebarMenuSkeleton`  | Loading skeleton                                                        |
| `SidebarTrigger`       | Toggle button                                                           |
| `SidebarRail`          | Slim rail for collapsed state                                           |
| `SidebarInset`         | Main content area next to sidebar                                       |
| `SidebarInput`         | Search input                                                            |
| `SidebarSeparator`     | Visual separator                                                        |

## Composable

`useSidebar()` — returns `{ state, open, setOpen, openMobile, setOpenMobile, isMobile, toggleSidebar }`.
