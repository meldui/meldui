## Sub-components

| Component             | Description                                |
| --------------------- | ------------------------------------------ |
| `Breadcrumb`          | Root `<nav>` element                       |
| `BreadcrumbList`      | Ordered list of breadcrumb items           |
| `BreadcrumbItem`      | Individual breadcrumb                      |
| `BreadcrumbLink`      | Clickable link. Props: `href`, `asChild`   |
| `BreadcrumbPage`      | Current page (non-clickable)               |
| `BreadcrumbSeparator` | Separator between items (default: chevron) |
| `BreadcrumbEllipsis`  | Collapsed items indicator                  |
