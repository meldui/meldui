## Usage

```vue
<script setup>
import {
  FileUpload,
  FileUploadDropzone,
  FileUploadTrigger,
  FileUploadList,
  FileUploadItem,
  FileUploadItemName,
  FileUploadItemSize,
  FileUploadItemDelete,
} from '@meldui/vue'
</script>

<template>
  <FileUpload accept="image/*" :max-files="5">
    <FileUploadDropzone>
      <p>Drag & drop files here, or</p>
      <FileUploadTrigger>Browse Files</FileUploadTrigger>
    </FileUploadDropzone>
    <FileUploadList>
      <template #default="{ files }">
        <FileUploadItem v-for="file in files" :key="file.name" :file="file">
          <FileUploadItemName />
          <FileUploadItemSize />
          <FileUploadItemDelete />
        </FileUploadItem>
      </template>
    </FileUploadList>
  </FileUpload>
</template>
```

## Sub-components

| Component                | Description                                                           |
| ------------------------ | --------------------------------------------------------------------- |
| `FileUpload`             | Root with file state management                                       |
| `FileUploadDropzone`     | Drag and drop area. Slot props: `isDragging`, `isDisabled`            |
| `FileUploadTrigger`      | Button to open file picker                                            |
| `FileUploadList`         | Container for file items                                              |
| `FileUploadItem`         | Individual file display. Props: `file`, `progress`, `status`, `error` |
| `FileUploadItemName`     | File name display                                                     |
| `FileUploadItemSize`     | File size display. Props: `precision`                                 |
| `FileUploadItemPreview`  | Image preview                                                         |
| `FileUploadItemProgress` | Upload progress bar. Props: `variant`                                 |
| `FileUploadItemDelete`   | Remove file button                                                    |
| `FileUploadClear`        | Clear all files button                                                |
