## Sub-components

| Component            | Description                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------ |
| `InputGroup`         | Root container                                                                                   |

| `InputGroupButton`   | Action button. Props: `variant`, `size`                                                          |
| `InputGroupInput`    | The input element                                                                                |
| `InputGroupText`     | Plain text element                                                                               |
| `InputGroupTextarea` | Textarea variant                                                                                 |
