## Usage

```vue
<script setup>
import { InputGroup, InputGroupAddon, InputGroupInput } from '@meldui/vue'
import { IconSearch } from '@meldui/tabler-vue'
</script>

<template>
  <InputGroup>
    <InputGroupAddon>
      <IconSearch class="size-4" />
    </InputGroupAddon>
    <InputGroupInput placeholder="Search..." />
  </InputGroup>
</template>
```

## With Button

```vue
<script setup>
import { InputGroup, InputGroupInput, InputGroupButton } from '@meldui/vue'
</script>

<template>
  <InputGroup>
    <InputGroupInput placeholder="Enter URL" />
    <InputGroupButton>Go</InputGroupButton>
  </InputGroup>
</template>
```

## Sub-components

| Component            | Description                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------ |
| `InputGroup`         | Root container                                                                                   |
| `InputGroupAddon`    | Icon/text addon. Props: `align` ('inline-start' \| 'inline-end' \| 'block-start' \| 'block-end') |
| `InputGroupButton`   | Action button. Props: `variant`, `size`                                                          |
| `InputGroupInput`    | The input element                                                                                |
| `InputGroupText`     | Plain text element                                                                               |
| `InputGroupTextarea` | Textarea variant                                                                                 |
