## Usage

```vue
<script setup>
import { Kbd, KbdGroup } from '@meldui/vue'
</script>

<template>
  <KbdGroup>
    <Kbd>⌘</Kbd>
    <Kbd>K</Kbd>
  </KbdGroup>
</template>
```

## Sub-components

| Component  | Description            |
| ---------- | ---------------------- |
| `Kbd`      | Individual key display |
| `KbdGroup` | Groups multiple keys   |
