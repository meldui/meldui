## Usage

```vue
<script setup>
import { ref } from 'vue'
import {
  TagsInput,
  TagsInputInput,
  TagsInputItem,
  TagsInputItemText,
  TagsInputItemDelete,
} from '@meldui/vue'

const tags = ref(['Vue', 'TypeScript'])
</script>

<template>
  <TagsInput v-model="tags">
    <TagsInputItem v-for="tag in tags" :key="tag" :value="tag">
      <TagsInputItemText />
      <TagsInputItemDelete />
    </TagsInputItem>
    <TagsInputInput placeholder="Add tag..." />
  </TagsInput>
</template>
```

## Sub-components

| Component             | Description                                                |
| --------------------- | ---------------------------------------------------------- |
| `TagsInput`           | Root with tag list state. Props from reka-ui TagsInputRoot |
| `TagsInputInput`      | Text input for adding new tags                             |
| `TagsInputItem`       | Individual tag. Props: `value` (required)                  |
| `TagsInputItemText`   | Tag text display                                           |
| `TagsInputItemDelete` | Remove button for the tag                                  |
