props:
  - name: orientation
    type: "'vertical' | 'horizontal' | 'responsive'"

    description: The layout direction of the field.
---

## Usage

```vue
<script setup>
import { Field, FieldLabel, FieldContent, FieldDescription, FieldError, Input } from '@meldui/vue'
</script>

<template>
  <Field>
    <FieldLabel>Username</FieldLabel>
    <FieldContent>
      <Input placeholder="Enter username" />
    </FieldContent>
    <FieldDescription>This is your public display name.</FieldDescription>
  </Field>
</template>
```

## With Error

```vue
<template>
  <Field>
    <FieldLabel>Email</FieldLabel>
    <FieldContent>
      <Input placeholder="Email" />
    </FieldContent>
    <FieldError :errors="['Please enter a valid email address.']" />
  </Field>
</template>
```

## Horizontal Layout

```vue
<template>
  <Field orientation="horizontal">
    <FieldLabel>Name</FieldLabel>
    <FieldContent>
      <Input placeholder="Name" />
    </FieldContent>
  </Field>
</template>
```

## Sub-components

| Component          | Description                                                 |
| ------------------ | ----------------------------------------------------------- |
| `Field`            | Root container. Props: `orientation`                        |
| `FieldLabel`       | Label text                                                  |
| `FieldContent`     | Input wrapper                                               |
| `FieldDescription` | Help text below the input                                   |
| `FieldError`       | Error message display. Props: `errors`                      |
| `FieldGroup`       | Groups multiple fields                                      |
| `FieldSet`         | Fieldset wrapper                                            |
| `FieldLegend`      | Legend for fieldset. Props: `variant` ('legend' \| 'label') |
| `FieldTitle`       | Section title                                               |
| `FieldSeparator`   | Visual separator between fields                             |
