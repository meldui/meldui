## Usage

```vue
<script setup>
import { ref } from 'vue'
import { Calendar } from '@meldui/vue'

const date = ref()
</script>

<template>
  <Calendar v-model="date" />
</template>
```

## Sub-components

Built on reka-ui CalendarRoot. All sub-components are exported for custom layouts: `CalendarHeader`, `CalendarHeading`, `CalendarGrid`, `CalendarGridHead`, `CalendarGridBody`, `CalendarGridRow`, `CalendarHeadCell`, `CalendarCell`, `CalendarCellTrigger`, `CalendarPrevButton`, `CalendarNextButton`.
