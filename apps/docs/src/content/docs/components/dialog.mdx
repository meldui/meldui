## Sub-components

| Component             | Description                                                              |
| --------------------- | ------------------------------------------------------------------------ |
| `Dialog`              | Root state container. Props: `open`, `defaultOpen`. Event: `update:open` |
| `DialogTrigger`       | Opens the dialog                                                         |
| `DialogContent`       | Modal content with overlay. Props: `showCloseButton`                     |
| `DialogScrollContent` | Scrollable content variant                                               |
| `DialogHeader`        | Header layout                                                            |
| `DialogTitle`         | Dialog title (required for accessibility)                                |
| `DialogDescription`   | Dialog description                                                       |
| `DialogFooter`        | Footer layout for actions                                                |
| `DialogClose`         | Closes the dialog                                                        |
| `DialogOverlay`       | Background overlay                                                       |
