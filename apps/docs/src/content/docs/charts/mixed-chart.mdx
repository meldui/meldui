## Config Reference

| Property   | Type                                                                   | Default  | Description                                         |
| ---------- | ---------------------------------------------------------------------- | -------- | --------------------------------------------------- |
| `series`   | `{ name: string; type: 'line' \| 'bar' \| 'area'; data: number[] }[]` | required | Each series specifies its own chart type             |
| `xAxis`    | `{ categories: string[] }`                                             | required | Category labels for the X axis                      |
| `stacked`  | `boolean`                                                              | `false`  | Stack bar/area series                               |
| `advanced` | `{ dualYAxis?: boolean; yAxisConfig?: object[] }`                      | -        | Advanced options including dual Y-axis support       |
