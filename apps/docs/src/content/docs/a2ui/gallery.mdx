Real, end-to-end surfaces an agent can build by streaming A2UI to the MeldUI catalog — ordered simple → complex. Each renders live; switch to the **Code** tab to see the exact **A2UI v0.9** message sequence (`createSurface` → optional `updateDataModel` → `updateComponents`) that produced it.

Want to tweak one? Copy its messages into the [Playground](/docs/a2ui/playground). For the per-component snippets, see the individual [component pages](/docs/a2ui/components/text); for the full contract, the [Catalog Reference](/docs/a2ui/reference).
