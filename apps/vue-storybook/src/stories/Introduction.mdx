## What's Included

- **UI Components** - Base components from shadcn-vue (Button, Card, Dialog, Table, etc.)
- **Composite Components** - DataTable, Timeline, MultiSelect, AvatarGroup, and more
- **Icon System** - 5000+ Tabler icons with custom size and stroke defaults
- **Charts** - Line, Bar, Pie, Area, Scatter, Radar, Heatmap charts
- **Theming** - OKLCH colors with CSS variables, light/dark mode support
- **TypeScript** - Full type safety across all components
- **Tailwind CSS v4** - Modern CSS-first configuration

## Quick Start

```bash
pnpm add @meldui/vue @meldui/tabler-vue vue
pnpm add -D tailwindcss @tailwindcss/vite tw-animate-css
```

```css
/* src/styles/app.css */
@import 'tailwindcss';
@import 'tw-animate-css';
@import '@meldui/vue/themes/default';

@source "../**/*.{vue,js,ts,jsx,tsx}";
@source "../node_modules/@meldui/vue/dist/**/*.mjs";
```

```vue
<script setup lang="ts">
import { Button, Card, CardContent } from '@meldui/vue'
import { IconCheck } from '@meldui/tabler-vue'
</script>

<template>
  <Card>
    <CardContent class="pt-6">
      <Button>
        <IconCheck class="mr-2 h-4 w-4" />
        Get Started
      </Button>
    </CardContent>
  </Card>
</template>
```
