# Example Pages

This section showcases complete page examples built using MeldUI components. These examples demonstrate how to compose multiple components together to create real-world applications.

## What You'll Find Here

Each example page represents a fully functional UI pattern that you can use as inspiration or a starting point for your own applications. They showcase:

- **Component Composition** - How to combine multiple components effectively
- **Layout Patterns** - Common page layouts and structures
- **Data Presentation** - Best practices for displaying various types of data
- **Real-World Use Cases** - Practical applications of the design system

## Available Examples

### Personal Financial Dashboard

A comprehensive dashboard showing personal finance data including:

- Portfolio overview with key metrics
- Asset allocation breakdown
- Holdings table with performance data
- Recent transactions
- Account summary

This example demonstrates the use of Cards, Tables, Badges, Progress indicators, and more in a cohesive layout.

## Using These Examples

These examples are meant to be:

1. **Learning Resources** - Study how components work together
2. **Copy & Paste Friendly** - Use the code as a starting point
3. **Customizable** - Adapt the layouts and data to your needs
4. **Interactive** - Play with the live examples using Storybook controls

Feel free to inspect the source code, copy snippets, and adapt them to your own projects.
